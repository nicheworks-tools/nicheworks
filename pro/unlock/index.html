How to use

  1. Copy the Pro key above.
  2. Open a supported tool such as JSON Repair.
  3. Paste the key into the local unlock field.
  4. Press Activate. The unlock state is saved in this browser.

This is a local browser unlock for supported NicheWorks tools. It does not send your tool input to a server.
