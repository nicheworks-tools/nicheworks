NicheWorks is a collection of small, quiet utilities designed to solve boring but recurring problems. No heavy dashboards, no accounts, no unnecessary data collection — just open, use, and close.

Tools

  • Unsubscribe Navi (JP)
    A directory of official cancellation pages for subscription services. (Japanese UI)
  • LogFormatter
    Browser-only formatter for Nginx / HTTP access logs. No upload, runs locally.
  • ManualFinder (JP)
    Links to official manual / documentation pages from major manufacturers. (Japanese UI)
  • More tools coming soon
    Additional English-ready utilities will be listed here.
