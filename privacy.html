プライバシーポリシー

NicheWorks（以下、「当サイト」）は、ユーザーが安心してツールを利用できるよう、 取り扱う情報や広告・解析に関する方針を以下の通り定めます。

1. 当サイトが収集する情報

当サイトの多くのツールは「ブラウザ内で完結」する設計であり、 ユーザーが入力したデータ（ファイル・テキスト等）はサーバーに送信されません。

ただし、以下の技術的情報はアクセス解析目的で収集する場合があります。

  • アクセスしたページ URL
  • アクセス日時
  • 利用デバイス・ブラウザ情報
  • 匿名化された IP アドレス（完全な個人特定は不可能）

※ Cloudflare Web Analytics を使用しており、個人を特定する情報は取得しません。

2. 広告について（Google AdSense）

当サイトでは、継続的な運営を目的として一部ページに Google AdSense（第三者配信の広告サービス） を利用しています。

AdSense は Cookie を使用し、ユーザーの興味に応じた広告を表示する場合があります。 Cookie を無効化したい場合はブラウザ設定より操作が可能です。

広告設定の詳細は以下をご参照ください：
https://policies.google.com/technologies/ads

3. 外部ツールの利用

当サイトでは以下の外部サービスを利用しています。

  • Cloudflare Web Analytics（匿名のアクセス解析）
  • Google AdSense（広告配信）

これらのサービスは匿名化されたデータを扱い、個人情報を特定するものではありません。

4. 個人情報の利用目的

当サイトがお問い合わせ等を通じてユーザーから任意に受け取った情報は、 以下の目的以外には使用しません。

  • ユーザーからの連絡への返信
  • 不具合報告・改善依頼への対応

5. 免責事項

当サイトで提供するツールは十分なテストの上公開していますが、 動作の完全性・正確性を保証するものではありません。

ツール利用により発生した損害について、運営者は一切の責任を負いません。 必要に応じて自己責任のもとご利用ください。

6. プライバシーポリシーの変更

本ポリシーは必要に応じて変更されることがあります。 重要な変更がある場合は当ページで告知します。

→ お問い合わせは Contact ページ からご連絡ください。
