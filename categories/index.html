PDF・書類

  • PDF to CSV Local
    PDF内の表データをCSV化します。
  • PDF Page Tools Mini
    PDFページの分割・並べ替え・削除を行います。
  • CSV Tidy
    CSVの列整理、ヘッダ名変更、空白トリムを行います。

画像・ファイル

  • EXIF Cleaner Mini
    画像メタデータを確認・削除します。
  • Image Redact
    画像の一部を隠します。
  • WebP / AVIF Converter
    WebP・AVIF画像をPNG/JPEGへ変換します。
  • Color Replace Lite
    画像内の指定色を置換します。
  • FileType Sniffer
    ファイル種別を推定します。

JSON・開発・安全

  • JSON Repair
    壊れたJSONを整えます。
  • JSON to Mermaid
    JSON構造をMermaid図へ変換します。
  • API Key Token Redactor
    APIキーやトークンらしき値を検出・マスクします。
  • Command Safety Checker
    実行前のコマンドに危険操作がないか確認します。

文章・仕事

  • LineBreak Doctor
    SNS投稿向けに改行を整形します。
  • Log Formatter
    ログを読みやすく整形します。
  • Rename Wizard
    ファイル名変更案を作ります。

生活・参照

  • ManualFinder
    公式マニュアル検索を補助します。
  • Laundry Code Decode
    洗濯表示を確認します。
  • UnitMaster
    単位を相互変換します。
  • 旧字体一覧
    旧字体と新字体の対応を確認します。
  • Construction Tools Atlas
    建設工具・現場用語を検索します。
