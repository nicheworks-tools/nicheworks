About

NicheWorks is a collection of small, quiet utilities designed to solve boring but recurring problems. No heavy dashboards, no accounts, no unnecessary data collection â€” just open, use, and close.

Tools

  • ManualFinder (JP)
    Link directory for official manuals / documentation pages for consumer devices.
  • TrashNavi (JP)
    Directory of official municipal waste-sorting / collection calendar pages in Japan.
  • LogFormatter
    Browser-only formatter for Nginx / HTTP access logs. No upload; everything runs locally.
  • Rename Wizard
    Preview-first filename normalizer for bulk renaming rules.
  • Contract Cleaner
    Highlights risky terms and phrases in Japanese contracts for a quick first-pass review.
  • Unsubscribe Navi (JP, external)
    Directory of official cancellation pages for subscription-type services. Hosted as a separate dynamic project.
