About This Site

NicheWorks is a collection of small web tools designed with quiet practicality in mind.
Rather than flashy features or buzz, the project values usability, simplicity, and long-term availability.
Ideas behind the project are summarized on the About page.

Tools

  • ManualFinder
    A directory of official manuals for home appliances, cameras, and consumer electronics.
  • TrashNavi
    Links to official municipal waste-sorting and collection calendars.
  • LogFormatter
    Format and search HTTP / Nginx access logs directly in the browser. No uploads required.
  • Rename Wizard
    Safely preview and copy batch file-name cleanup rules.
  • Contract Cleaner
    Highlights risky or concerning keywords in contract documents.

Policy

Many of the tools are designed to work entirely inside your browser.
No input data is sent to the server unless explicitly stated.
