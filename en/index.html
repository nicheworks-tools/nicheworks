Old Japanese kanji reference

NicheWorks provides a reference page for comparing old Japanese kanji forms with their modern equivalents. It is useful when checking names, place names, older books, old signs, historical documents, and family records.

Main published page

  • Old Kanji Reference
    Search old Japanese kanji forms and compare them with modern equivalents. Selected entries include readings, meanings, usage notes, and categories.

How to use

  • Search
    Filter entries by old form, modern form, and selected readings or meanings.
  • Compare forms
    Some modern forms can correspond to multiple old or variant forms, so the page helps compare them side by side.
  • Copy characters
    Copy displayed characters for notes, research, or document preparation.
