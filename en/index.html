Free browser tools

NicheWorks is a collection of small web tools for PDF, images, CSV, JSON, text cleanup, privacy checks, and everyday work. Many tools run directly in your browser and do not store entered text or files on NicheWorks servers.

How this site is operated

  • Small free utilities
    NicheWorks publishes focused browser tools for everyday tasks, document cleanup, privacy checks, and developer support.
  • Local-first processing
    Whenever possible, text, image, CSV, JSON, and document handling happens in the browser. Tools that need external communication explain it on their pages.
  • Ads and optional support
    Some pages may display Google AdSense ads. Optional support links are also available to help maintain the project.

Main tools

The main tools are listed as static HTML so users, crawlers, and JavaScript-disabled browsers can read the site content directly.

PDF, CSV, and JSON

  • JSON Repair
    Repair, validate, and format broken JSON locally in your browser.
  • JSON to Mermaid
    Convert JSON structures into Mermaid diagrams for easier documentation.
  • PDF to CSV Local
    Extract table-like PDF data into CSV-style output for local review.
  • PDF Page Tools Mini
    Split, reorder, and remove PDF pages in a lightweight browser workflow.
  • CSV Tidy
    Reorder columns, rename headers, trim whitespace, and clean CSV text.

Images, files, and privacy

  • EXIF Cleaner Mini
    Check and remove image metadata before sharing photos or screenshots.
  • Image Redact
    Hide sensitive parts of screenshots with blur, mosaic, or black boxes.
  • FileType Sniffer
    Inspect file signatures and estimate file types before sharing or opening.
  • API Key Token Redactor
    Detect and mask API keys, tokens, and secrets before sharing text or logs.
  • Command Safety Checker
    Check shell commands for risky operations before running them.

Reference and everyday helpers

  • ManualFinder
    Help find official manuals and support pages from product or brand names.
  • Old Kanji Reference
    Search old Japanese kanji forms and compare them with modern equivalents.
  • Kanji Modernizer
    Convert or compare old Japanese kanji forms with modern forms.
  • Laundry Code Decode
    Decode laundry symbols and care-label notes.
  • UnitMaster
    Convert common units such as length, weight, temperature, and volume.

Ads and data handling

NicheWorks may display ads to support operation. Details about Google AdSense, Google Analytics 4, Cloudflare Web Analytics, cookies, and tool input data are explained in the Privacy Policy. For questions, use the Contact page.

Support

If a tool helped you, optional support is appreciated and helps maintain the project.
