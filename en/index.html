Free browser tools

NicheWorks is a collection of small tools for PDF, images, CSV, JSON, text cleanup, privacy checks, and everyday work. Many tools run locally in your browser without storing your input.

Featured tools

These fixed links are kept in the HTML so both users and search engines can reach the main tools even before JavaScript renders the full list.

  • JSON Repair
    Repair, validate, and format broken JSON locally in your browser.
  • PDF to CSV Local
    Extract table-like PDF data into CSV-style output locally.
  • CSV Tidy
    Reorder, rename, trim, and clean CSV columns in the browser.
  • EXIF Cleaner Mini
    Remove image metadata before sharing photos or screenshots.
  • Image Redact
    Hide sensitive parts of screenshots with blur, mosaic, or black boxes.
  • API Key Token Redactor
    Detect and mask API keys, tokens, and secrets before sharing text or logs.
  • Command Safety Checker
    Check shell commands for risky operations before running them.
  • ManualFinder
    Find official manuals and support pages from product or brand names.
  • Construction Tools Atlas
    Browse construction tools, aliases, uses, and work categories.
  • FileType Sniffer
    Inspect file signatures and estimate file types locally.
