ツールを開く English guide

こんなときに使う

  • 現場で聞いた言葉が分からず、ざっと意味の当たりを付けたい
  • 英語圏の施工動画・マニュアルで出てくる用語を調べたい
  • 道具名・工程名をカテゴリから探して覚えたい

関連キーワード：construction slang / tool name / job process / 現場用語 / 工具 名称 / 工程 用語

使い方

  1. 検索欄にキーワードを入力（例：tool / process / slang など）
  2. カテゴリで絞り込んで、近いエントリを探す
  3. 必要なら言語トグル（JA/EN）で表示を切り替える

コツ：最初は “道具名（tool）” か “作業名（process）” で当たりを付けると早い。

注意（できないこと・限界）

  • 地域・会社・職種で呼び方が変わることがあります（同義語が複数ある前提）。
  • 法令・安全基準・施工手順の正しさを保証するツールではありません。
  • 辞典データは更新中のため、未収録・簡略説明の項目がある場合があります。
