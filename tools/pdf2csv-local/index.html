詳しい説明: 使い方 / 変換のコツ / FAQ

Extract tables from text-based PDFs and save the result as CSV or Excel. This is for statements, invoices, and tabular lists.

Docs: Usage / How-to / FAQ

対応PDFについて

向いているPDF

  • 文字を選択できるPDF
  • 銀行明細、請求書、一覧表
  • 行と列がある程度そろっている表

向いていないPDF

  • 画像スキャンPDF・写真PDF
  • OCRが必要なPDF
  • 斜め、複数段、装飾が多い表
  • パスワード付きPDF・暗号化PDF

このツールはOCRを行いません。文字を選択できないPDFは抽出できません。

PDF内容は外部送信されません。ただし、PDF解析ライブラリの読み込みに外部CDNを使用する場合があります。

Supported PDFs

Good fit

  • PDFs where text can be selected
  • Bank statements, invoices, and lists
  • Tables with reasonably aligned rows and columns

Not a good fit

  • Scanned image PDFs or photo PDFs
  • PDFs that require OCR
  • Rotated, multi-column, or heavily decorated tables
  • Password-protected or encrypted PDFs

This tool does not run OCR. PDFs without selectable text cannot be extracted.

PDF contents are not uploaded. The PDF parsing library may be loaded from an external CDN.

よくある質問

画像スキャンPDFをCSVにできますか？
いいえ。OCRには対応していません。文字を選択できるPDFが対象です。
PDFはサーバーに送信されますか？
PDF内容は外部送信されません。ブラウザ内で処理します。ただしPDF解析ライブラリの読み込みに外部CDNを使用する場合があります。
抽出結果は正確ですか？
PDFのレイアウトに依存します。必ずプレビューを確認してください。
Excel形式で保存できますか？
はい。抽出結果をXLSX形式で保存できます。ただしPDFの見た目を完全再現するものではありません。
CSVが文字化けします。
Excelで開く場合はCSV BOMをONにしてください。

FAQ

Can this convert scanned image PDFs to CSV?
No. OCR is not supported. Use PDFs with selectable text.
Are PDFs uploaded to a server?
No PDF contents are uploaded. Processing runs in your browser. The PDF parsing library may be loaded from an external CDN.
Is extraction always accurate?
No. It depends on the PDF layout. Always review the preview.
Can I save as Excel?
Yes. It saves the extracted table as XLSX, but it does not recreate the original PDF layout.
The CSV is garbled in Excel.
Turn on CSV BOM when opening the file in Excel.
