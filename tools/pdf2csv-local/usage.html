使い方

PDF2CSV Localの操作手順を簡潔にまとめます。

/

できること

  • 銀行明細・請求書などの表をPDFからCSV/XLSXに変換
  • Autoで自動抽出、Manualで範囲指定して再抽出
  • 複数ページの表をまとめて出力

手順（Auto / Manual）

Auto

  1. PDFを読み込み、ページ範囲と出力形式を選ぶ
  2. 抽出設定（行・列の近接判定）を必要に応じて調整
  3. Extractを実行してプレビューを確認し、CSV/Excelを保存

Manual

  1. モードをManualに切り替える
  2. プレビュー上で表の範囲をドラッグ選択
  3. Re-extractで再抽出する
  4. 結果を確認してCSV/Excelで保存

対応/非対応

対応

  • テキスト情報を含むPDF（一般的な明細・請求書）
  • 複数ページの連続した明細

非対応

  • 画像スキャンのみのPDF（文字が画像として埋め込まれているもの）
  • 強い暗号化・パスワード付きPDF
  • 複雑なレイアウトで表が分断されているPDF

トラブルシュート

  • 行がまとまらない → 行の近接判定を少し大きくする
  • 列が結合される → 列のギャップ判定を少し大きくする
  • 抽出が少ない → Manualで範囲指定して再抽出
  • ページ範囲エラー → 入力形式を「1-3,5」などに合わせる

出力形式

  • CSV: 複数ページは区切り行（=== Page n ===）を挿入
  • Excel: ページごとにシートを作成（Page1, Page2...）
  • BOM付与や区切り文字はプレビューで変更可能

免責

変換精度は保証しません。処理はブラウザ内で完結し、PDFは外部に送信されません。
