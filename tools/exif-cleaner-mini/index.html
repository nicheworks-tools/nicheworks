このツールの仕組み

How this tool works

画像をブラウザ内でCanvasに描き直し、新しい画像ファイルとして保存します。その過程で多くのEXIFメタデータは引き継がれません。

This tool redraws the image in a browser canvas and saves it as a new image file. Most EXIF metadata is not carried over during this process.

ただし、すべてのアプリ独自メタデータやファイル構造情報の削除を保証するものではありません。

It does not guarantee removal of every app-specific metadata field or file-level trace.

削除される可能性のある情報

Metadata that may be removed

  • GPS位置情報
  • GPS location data
  • 撮影日時
  • Capture date & time
  • カメラ/端末モデル
  • Camera / device model
  • レンズ/ソフト情報
  • Lens / software information

※ 本ツールはマルウェア検査ではありません。メタデータ削除用の簡易ツールです。

* This is not malware scanning. It is a simple metadata-cleaning tool.

※ すべての処理はブラウザ内で実行されます（画像は送信されません）。

* All processing runs locally in your browser. No images are uploaded.

よくある質問

FAQ

画像はサーバーに送信されますか？

Are images uploaded to a server?

いいえ。処理はブラウザ内で行われます。

No. Processing runs in your browser.

GPS位置情報は削除されますか？

Does it remove GPS location data?

Canvasで画像を再生成するため、多くの場合GPS情報は引き継がれません。ただし完全保証ではありません。

Because the image is regenerated through canvas, GPS data is usually not carried over. This is not a full guarantee.

EXIFが見つからない画像でも処理する意味はありますか？

Is it useful even if no EXIF is detected?

あります。アプリによってはEXIF以外の情報が含まれる場合があるため、共有前に再生成しておく用途があります。

Yes. Some apps may add non-EXIF metadata, so regenerating the image before sharing can still be useful.

HEICに対応していますか？

Does it support HEIC?

現時点では JPEG / PNG / WebP を対象にします。HEICは先にJPEGまたはPNGへ変換してください。

Currently it supports JPEG, PNG, and WebP. Convert HEIC to JPEG or PNG first.
