Remove GPS/time/device EXIF locally in your browser before you share photos.

What it does

  • Removes EXIF metadata (GPS, timestamps, device info) from photos
  • Designed to run locally in your browser (no storage)
  • Lightweight and mobile-friendly
Open the tool 日本語ガイド

Use this when

  • You want to remove GPS location before posting to social media
  • You’re sharing photos for work but don’t need device/time info
  • You took photos near home and want to avoid location leakage

Keywords: remove EXIF, remove GPS metadata, photo metadata remover, strip EXIF

How to use

  1. Open the tool and select a photo (or drag & drop)
  2. Run EXIF removal when prompted
  3. Download the cleaned image and share that version

Tip: verify on another app/device if you want extra confidence.

Notes (limits)

  • This does not remove visual information in the image (background/address in the photo).
  • Some formats/special metadata may not be fully removed.
  • The original file remains unchanged unless you replace it manually.
