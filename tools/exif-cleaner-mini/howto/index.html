EXIF Cleaner Mini：使い方ガイド

写真のGPS位置情報・撮影日時・端末名などを、共有前に減らすためのブラウザ完結ツールです。

広告枠

このツールでできること

  • JPEG / PNG / WebP画像をブラウザ内で再生成します。
  • Canvasに描き直して保存することで、多くのEXIFメタデータを引き継がない画像を作ります。
  • 画像はサーバーへ送信されません。
ツールを開く 詳しい注意点を見る English guide

使い方

  1. ツールを開き、JPEG / PNG / WebP画像を選択します。
  2. 簡易チェック結果を確認します。この判定はGPS有無の詳細解析ではありません。
  3. 必要に応じて出力形式やJPEG画質を選びます。
  4. 「EXIFを削除して保存」を押します。
  5. -cleaned が付いた保存後画像を共有用として使います。

注意点

  • すべてのアプリ独自メタデータやファイル構造情報の削除を保証するものではありません。
  • 写真の中に写っている住所、顔、注文番号、画面上の名前などは消えません。
  • HEIC、TIFF、RAW、PDF、動画には対応していません。
  • 元画像は変更されません。必要なら自分で元画像を削除してください。

あわせて使うツール

  • Image Redact：画像内の名前・住所・番号などを黒塗りします。
  • FileType Sniffer：実際のファイル形式を確認します。
  • WebP / AVIF Converter：画像形式を変換します。
