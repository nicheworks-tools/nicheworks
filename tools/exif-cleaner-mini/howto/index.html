EXIF Cleaner Mini：使い方

写真の位置情報や撮影情報（EXIF）を、アップロードせずに削除してから共有できます。

広告枠（準備中）

このツールでできること

  • 写真に含まれる EXIF メタデータ（GPS / 撮影日時 / 端末情報など）を削除
  • ブラウザ内で処理（外部送信なし想定）
  • スマホでも使いやすい軽量版
ツールを開く English guide

こんなときに使う

  • SNSやメッセージで写真を送る前に、位置情報が入っていないか不安
  • 仕事で写真を共有するが、端末情報や撮影日時は不要
  • 家の周辺で撮った写真を公開したいが、GPSが怖い

関連キーワード：EXIF 削除 / 写真 位置情報 消す / GPS metadata remover / remove photo metadata

使い方

  1. ツールを開き、写真ファイルを選択（またはドラッグ）
  2. プレビューや表示が出たら、EXIF削除を実行
  3. 保存された画像（EXIFなし）をダウンロードして共有する

コツ：共有前に、別の端末/アプリで位置情報が消えているか確認するとさらに安心。

注意（できないこと・限界）

  • EXIF以外の情報（画像そのものの写り込み・背景の住所など）は消えません。
  • 一部形式/特殊なメタデータは完全に削除できない場合があります。
  • 編集後の画像は別名保存されるため、元画像（EXIFあり）はそのまま残ります。
