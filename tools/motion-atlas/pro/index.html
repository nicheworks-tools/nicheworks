Motion Atlas Pro plan

The free version must remain useful. Pro is for deeper comparison, export, and framework-specific work after the core tool is stable.

Ad space

No paid lock, payment, login, server saving, or Pro detection is wired into the initial free version.

The current tool should stand alone without Pro. Pro is a later expansion layer.

Free version stays core

  • Live demos
  • Search and filters
  • Detail panel
  • Compare up to 2
  • Favorites and recent history
  • Prompt copy and basic CSS hints
  • Replay, pause, speed, and reduced-motion controls

Future Pro candidates

  • 3+ motion comparison
  • Saved compare sets
  • React / Vue / Tailwind / Framer Motion snippets
  • Detailed parameter tuning
  • Prompt bundles for common product tasks
  • Print and export views
  • Project-specific saved motion sets

Boundary

Pro must not weaken the free catalog. It should add workflow depth for people who already use the free tool, not hide the basic motion dictionary behind a paywall.
