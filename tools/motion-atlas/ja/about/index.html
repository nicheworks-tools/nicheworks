広告枠

Motion Atlas は、「いい感じに動かして」では伝わらないUIモーションを、名前・見た目・使いどころ・非推奨用途・AI依頼文まで含めて整理する辞典です。

UI Atlas / Vibe Lexicon と同じ NicheWorks Atlas 系ツールとして扱います。

何のためのツールか

モーション名を知らない、似た動きの違いが分からない、AIにどう頼めばよいか分からない状態を減らすための制作支援ツールです。

何ではないか

医療ツール、アクセシビリティ監査、ブラウザ互換性保証、完成コード集ではありません。実装の出発点と判断補助として使います。

UI Atlas / Vibe Lexicon との違い

UI Atlas はUIパターン、Vibe Lexicon は雰囲気や見た目の言語化、Motion Atlas は「どう動くか」「なぜその動きが必要か」「どこでは使わないか」を扱います。

アクセシビリティ方針

モーションは状態・位置・反応の理解を助けますが、負担になる場合もあります。そのため各パターンに reduced motion の代替とループ抑制を前提にします。
