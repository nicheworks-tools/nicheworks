Motion decision memo

Selected motion、Why this motion、When to use、When not to use、Risk、Fallback、Implementation note、Review checklist。

Framer Motion prompt

component target、initial / animate / transition、duration、easing、reduced motion対応、禁止する動き。

CSS implementation prompt

CSS keyframes、class設計、duration、easing、reduced-motion media query、replay設計。

Tailwind / React prompt

React component前提、Tailwind class、state / replay、reduced motion、accessibility注意。

Codex task

対象ファイル、実装目的、変更内容、acceptance criteria、reduced motion要件、regression check。

GitHub Issue

title、background、implementation requirements、accessibility requirements、acceptance criteria、test checklist。

reduced motion checklist

prefers-reduced-motion、transform削除、loop停止、shake / pulse / bounce代替、autoplay抑制、replay、pause、focus影響。

compare handoff

Compared motions、Recommendation、Reason、Rejected alternatives、Fallback、Implementation note、Codex handoff。

購入後の流れ

  1. 共通Payment Linkで購入します。
  2. Stripe完了後に /pro/unlock/?session_id={CHECKOUT_SESSION_ID} へ戻ります。
  3. /assets/nw-pro.js が /api/pro/status を確認します。
  4. サーバー側の正はD1 pro_entitlements の nicheworks_pro です。
  5. このブラウザに補助状態を保存し、Motion Atlasは NWPro.getLocalStatus() を読みます。

購入後、このブラウザではNicheWorks Proが有効になります。タブやブラウザを閉じても通常は維持されます。ただし、別端末・別ブラウザ・シークレットモード・サイトデータ削除後は再度有効化が必要です。

制限事項と共通Pro基盤

  • localStorageは補助状態です。サーバー側の正はD1 pro_entitlements です。
  • Motion Atlasは共通クライアント /assets/nw-pro.js と共通entitlement nicheworks_pro を使います。
  • Motion Atlas用に別Stripe商品、ツール専用Webhook、ツール専用D1ストアは作りません。
  • 見る、探す、動かす、reduced motionを見る、2件比較、基本依頼文コピーは無料のままです。
