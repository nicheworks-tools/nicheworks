広告枠

初期無料版には、決済、課金ロック、ログイン、サーバー保存、Pro判定を入れません。

現在のツールはProなしで成立させます。Proは後から足す拡張層です。

無料版に残す中核

  • 実際に動くデモ
  • 検索とフィルタ
  • 詳細パネル
  • 2件比較
  • お気に入りと最近見た項目
  • AI依頼文コピーと基本CSSヒント
  • 再生、一時停止、速度、reduced motion操作

将来のPro候補

  • 3件以上の比較
  • 比較セット保存
  • React / Vue / Tailwind / Framer Motion 向けコード
  • 詳細パラメータ調整
  • 用途別prompt bundle
  • 印刷・書き出し
  • プロジェクト別の保存セット

境界

Proは無料カタログを弱くするためのものではありません。無料版で辞典として使えた上で、実務フローを深めるための追加機能に限定します。
