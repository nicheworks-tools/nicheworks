FAQ

Does this call an AI API?

No. This page is static and client-side only. The previews are explanatory UI examples, not real AI output.

What is free?

All patterns, search, filters, detail view, best for / not for, failure states, trust notes, implementation notes, common mistakes, two-item comparison, recent history, five favorites, related links, and basic prompt copy are free.

What is Pro for?

Pro converts the selected AI UI pattern into Product spec handoff, Codex task, GitHub Issue, UX risk checklist, Safety / fallback checklist, Markdown / JSON exports, and 3–4 pattern comparison.

What happens after purchase?

After purchase, NicheWorks Pro becomes active in this browser. It normally remains available after closing tabs or the browser. Another device, another browser, private browsing, or deleted site data requires activation again.
