出力サンプル

Previewモード。

無料機能は引き続き利用できます。共通Proを有効にすると、メインツールでこれらの出力をコピー/exportできます。

Pro解放済み。

このブラウザでは共通Proが有効です。メインツールで選択中パターンからコピー/export可能な出力を生成してください。

Product spec handoff

pattern name、Japanese label、category、purpose、AI risk、user control、AI visibility、trust notes、best/not-for、required UI states、fallback behavior、acceptance criteria。

Codex実装依頼文

実装対象、変更対象ファイルの想定、UI states、accessibility、trust要件、free behavior、pro behavior、no AI API call、no external data send、acceptance criteria。

GitHub Issue形式

summary、scope、implementation requirements、free/Pro behavior、UI states、accessibility requirements、QA checklist、out of scope。

UX risk checklist

AIが正しく見えすぎないか、AI生成であることが分かるか、出典・根拠・制限が見えるか、人間確認が明確か、編集・拒否・取り消しができるか、失敗時に詰まらないか、高リスク操作を自動適用していないかを確認します。

Safety / fallback checklist

empty、loading、partial、no useful answer、refusal、rate limit、source unavailable、manual fallback、retry、edit and retry、undo、human handoff。

Markdown / JSON export

選択中patternのPro出力一式、または比較結果のhandoff packをMarkdown / JSONとして保存できます。

購入と有効化

購入後、このブラウザではNicheWorks Proが有効になります。タブやブラウザを閉じても通常は維持されます。ただし、別端末・別ブラウザ・シークレットモード・サイトデータ削除後は再度有効化が必要です。

このツールは共通クライアント /assets/nw-pro.js と NWPro.getLocalStatus() で共通Pro状態を読みます。entitlement は nicheworks_pro です。localStorageは補助状態で、サーバー側D1の pro_entitlements が正です。

AI Interaction Atlas 用の個別Stripe商品、個別Payment Link、個別Webhook、個別D1、個別entitlementは作りません。
