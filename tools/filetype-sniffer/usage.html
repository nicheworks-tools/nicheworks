FileType Sniffer — 使い方

FileType Sniffer は、ファイル先頭のMagic Numberをブラウザ内で読み取り、本当の形式を推定する無料ツールです。

このツールは形式推定のみを行います。安全性、マルウェア、実行可否は判定できません。出所不明なファイルは開かないでください。

1. このツールでできること

  • ファイル先頭バイトから形式を推定します。
  • 拡張子と実際の形式が違う可能性を警告します。
  • DOCX / XLSX / PPTX / APK / JAR / EPUB などのZIPコンテナ系を補足表示します。
  • EXE / ELF / Mach-O / APK / JAR など実行・インストール系の注意を表示します。

2. 使い方の手順

  • 判定したいファイルをドラッグ＆ドロップ、またはファイル選択から選びます。
  • 自動で解析されます。必要なら「解析する」を押し直します。
  • サマリーとJSONで、判定形式・MIME type・信頼度・検出バイトを確認します。
  • 拡張子不一致や実行系の警告が出た場合は、出所不明なら開かないでください。

3. 対応 / 非対応

主な対応形式：

  • PDF / PNG / JPEG / GIF / WebP / AVIF / HEIC / HEIF
  • ZIP / 7Z / GZIP / RAR
  • MP3 / WAV / MP4 / MOV / WebM / MKV
  • BMP / TIFF / ICO
  • EXE / ELF / Mach-O

非対応・注意：

  • ファイル全体の中身や安全性は確認しません。
  • ZIP内部ファイル名の完全解析は行いません。ZIP派生の補足は主に拡張子ベースです。
  • 暗号化・独自ヘッダ・破損ファイルは判定できない場合があります。

4. エラーについて

  • ファイル未選択：ファイルを選んでから解析してください。
  • 読み取りエラー：ブラウザがファイルを読めない状態です。
  • Unknown：既知のMagic Numberに一致しなかった状態です。

5. 出力内容の説明

  • 読み取り量：先頭最大4096バイトのみを読み取ります。
  • 判定形式：Magic Numberから推定した形式です。
  • MIME type：推定形式に対応する代表的なMIMEです。
  • 一致判定：拡張子と推定形式の関係です。
  • 次に確認すること：開く前に注意すべき点です。

6. よくある質問

Q. ウイルスを検出できますか？
A. いいえ。形式推定のみで、安全性は判定できません。

Q. DOCXやXLSXがZIP扱いになるのはなぜですか？
A. Office文書の一部はZIPコンテナ構造を使っているためです。

Q. ファイルはアップロードされますか？
A. いいえ。ブラウザ内で先頭バイトだけを読み取ります。

7. 免責

本ツールはMagic Numberに基づく形式推定のみを行います。判定結果や安全性は保証しません。重要なファイルや不明なファイルは、信頼できる方法で確認してください。
