FileType Sniffer — 使い方

このページでは、FileType Sniffer の基本的な使い方や対応形式、エラーの見方をまとめています。

1. このツールでできること

FileType Sniffer は、ファイルの先頭バイト（Magic Number）を解析し、 「本当のファイル形式」 を推定する無料ツールです。 拡張子偽装のチェックや、壊れたファイルの形式確認に役立ちます。

2. 使い方の手順

  • ① 判定したいファイルをドラッグ＆ドロップ、または「ファイルを選択」から選びます。
  • ② 「解析する」ボタンを押します。
  • ③ 先頭の Magic Number を読み取り、推定される形式を表示します。
  • ④ 判定ができない場合は「不明」と表示されます。

3. 対応 / 非対応

対応している主な形式：

  • PNG / JPEG / GIF / WebP
  • PDF
  • ZIP / RAR / 7Z
  • MP4 / MOV / AVI / MKV
  • BMP / TIFF / ICO
  • WAV / MP3
  • EXE / ELF

非対応の例：

  • 巨大なファイル（1GB以上）
  • 複雑な暗号化・独自ヘッダを持つ特殊形式
  • 破損が激しく先頭バイトが読めないファイル

4. エラーについて

よくあるエラー：

  • ファイルが選択されていない → ファイルを選んでから解析してください。
  • 読み取りエラー → ファイルがロックされている／破損している可能性があります。
  • Magic Number が存在しない → 判定できない形式です。

5. 出力内容の説明

解析結果には以下が含まれます：

  • 推定形式（例：PNG / ZIP / MP4）
  • ファイル名
  • サイズ（bytes）
  • MIME（ブラウザ判定）
  • Magic Number（先頭の Hex 値）

6. よくある質問（FAQ）

Q. Magic Number とは何ですか？
A. ファイルの先頭にある形式識別用のバイト列で、拡張子より信頼できます。

Q. 推定形式は100%正確ですか？
A. 主な形式は正確に判定できますが、Magic Number が共通の形式では誤判定もあり得ます。

Q. 拡張子が違うのに形式が一致するのはなぜ？
A. 拡張子偽装またはリネームされたファイルの可能性があります。

Q. 危険なファイルかどうか確認できますか？
A. 形式の推定はできますが、安全性の保証はできません。

7. 免責

本ツールは Magic Number に基づく形式推定のみを行います。 判定結果の正確性は保証できません。安全性の判断は自己責任で行ってください。
