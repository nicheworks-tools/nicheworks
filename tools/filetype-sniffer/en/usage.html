FileType Sniffer — How to Use

This page explains how FileType Sniffer works and how to interpret its results and errors.

1. What this tool does

FileType Sniffer analyzes the first bytes of a file — known as Magic Numbers — to estimate the file’s true format. This is useful for checking extension mismatches and inspecting suspicious or corrupted files.

2. Steps to use

  • ① Drag & drop a file, or select it manually.
  • ② Press the “Analyze” button.
  • ③ The tool reads the Magic Number and shows the estimated format.
  • ④ If detection fails, “Unknown” will be shown.

3. Supported / Unsupported

Supported formats include:

  • PNG / JPEG / GIF / WebP
  • PDF
  • ZIP / RAR / 7Z
  • MP4 / MOV / AVI / MKV
  • BMP / TIFF / ICO
  • WAV / MP3
  • EXE / ELF

Unsupported cases:

  • Very large files (1GB+)
