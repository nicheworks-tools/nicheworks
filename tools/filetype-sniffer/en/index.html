FAQ

Q. What is a Magic Number?
A. It is a sequence of bytes at the beginning of a file used to identify its type.

Q. Is the detection 100% accurate?
A. Not always. Some formats share identical Magic Numbers.

Q. Why does the file extension differ from the detected type?
A. The file may have been renamed or intentionally disguised.

Q. Can this tool check whether a file is safe?
A. No. It only identifies the format and does not verify file safety.
