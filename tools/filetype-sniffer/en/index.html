FAQ
Q. Can this tool perform a security scan?
A. No. It only estimates the file format and cannot determine whether a file is safe.
Q. Is it risky if the extension and detected type differ?
A. It may indicate manual renaming or extension spoofing. Avoid handling files from unknown sources.
Q. Why are DOCX or XLSX files detected as ZIP?
A. Some Office documents use a ZIP-based container structure. APK, JAR, EPUB, and OpenDocument files may also be ZIP-based.
Q. Are files uploaded?
A. No. The tool reads only the first bytes inside your browser.
