FileType Sniffer：使い方

拡張子ではなく “中身（Magic Number）” から本当のファイル形式を判定します。

広告枠（準備中）

このツールでできること

  • ファイル先頭の識別情報（Magic Number / Signature）を読み取り、形式を推定
  • 拡張子偽装（.jpgに見せかけた別形式など）の確認に使える
  • 壊れたファイルで「そもそも何形式か」を確認する補助になる
  • ブラウザ完結・保存なし
ツールを開く English guide

こんなときに使う

  • 拡張子が怪しい／中身と一致しているか不安
  • ファイルが開けないが、形式だけでも知りたい
  • 受け取ったファイルが安全か確認したい（第一段階）

関連キーワード：magic number 判定 / 拡張子 偽装 チェック / file signature / file type detector

使い方

  1. ツールを開く
  2. 判定したいファイルを選択（またはドラッグ）
  3. 表示された推定形式とシグネチャを確認する

コツ：結果が想定と違う場合は、拡張子を手動で合わせると開けることがあります（自己責任）。

注意（できないこと・限界）

  • 判定は「推定」です。複合形式・暗号化・一部破損などでは誤判定する場合があります。
  • このツール単体で安全性（マルウェア等）を保証するものではありません。
  • 圧縮ファイル内の中身までは解析しません（対応していない場合）。
