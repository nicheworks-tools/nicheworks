How to Use

  1. 崩れた文章を下の入力欄に貼り付けます。
  2. 「整形する」を押します。
  3. SNS別に最適化したテキストが下に表示されます。
  4. 使いたいSNSの「コピー」を押して投稿に貼ります。
  1. Paste your text into the input area below.
  2. Click “Format”.
  3. Optimized text for each SNS appears below.
  4. Press “Copy” on the SNS you want to post to.
