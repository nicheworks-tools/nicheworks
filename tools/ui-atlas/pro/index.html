What Free can do

Free is the basic UI pattern reference for browsing and checking common interface patterns before you commit to an implementation.

  • View 100 UI examples.
  • Search UI patterns, browse cards, and open detail views.
  • Use live samples and Short AI prompt copy.
  • Keep local favorites and recently viewed items.
  • Compare up to 2 patterns.
  • See Pro-only 50 samples as locked preview cards.
  • See part of the Pro output format as Preview.

After purchase

After purchase, Stripe returns to /pro/unlock/?session_id={CHECKOUT_SESSION_ID}. The unlock page checks the shared D1-backed nicheworks_pro entitlement and stores browser helper state for supported tools.

NicheWorks Pro usually remains active in the same browser after closing the tab or browser. Another browser, another device, private mode, or clearing site data may require unlocking again. localStorage is only a browser helper; D1 pro_entitlements is the server-side source of truth.
