Current status

There is no active paid lock in UI Atlas right now. Free features remain open, and this page is a transparent direction note rather than a fake checkout surface.

Potential Pro value (future)

  • Deeper compare dimensions (for example: accessibility risk profiles, anti-pattern warnings, and implementation caveats by context).
  • Expanded prompt kits that map pattern decisions to build-ready request templates.
  • Saved compare sets and decision logs for teams that repeat similar UI choices.

What will not happen

  • No fake “locked” toggles that do nothing.
  • No pretend billing controls before real billing infrastructure exists.
  • No sudden removal of current free core workflows without clear announcement.

Why this roadmap still matters now

Even before billing, roadmap clarity helps teams decide whether UI Atlas can become part of their long-term reference workflow. Trust comes from clear boundaries and honest status updates.
