Pro is for people who need to ship, not just browse

Free UI Atlas helps you find the right UI pattern. Pro adds the missing step: turning that pattern into a practical spec, AI build request, and review checklist.

Unlock Pro — $2.99

What you get with Pro

  • UI decision memo: copy-ready wording for selected pattern, rejected alternatives, reason, risks, and validation plan.
  • Advanced compare checklist: accessibility risk, mobile fit, content-length risk, implementation cost, maintenance cost, and misuse warnings.
  • AI/Codex handoff prompts: longer prompts that include role, context, constraints, acceptance criteria, and fallback behavior.
  • Implementation review checklist: keyboard access, focus state, reduced motion, empty state, error state, loading state, and responsive checks.
  • Export-ready wording: Markdown-style text for GitHub issues, design docs, product specs, and small client handoffs.

Free vs Pro

  • Free: 50 live UI pattern examples, search, filters, detail panel, short prompt, favorites, recent history, and 2-item quick compare.
  • Pro: decision and handoff materials that make the catalog useful after you choose a pattern.

Sample Pro output

Decision: Use a confirmation dialog for destructive account deletion because the action is irreversible and requires explicit user intent.

Reject: Do not use a toast or inline warning alone, because it can be missed and does not force confirmation.

Implementation checklist: clear title, consequence copy, cancel button first, destructive button visually distinct, keyboard focus trapped, Escape closes, screen reader label present.
