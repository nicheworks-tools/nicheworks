English
ホーム使い方このツールについてPro

Proは、眺める辞典を「実装に渡せる材料」にするための機能です

無料版UI Atlasは、適切なUIパターンを探して比較するための辞典です。Proでは、その選択を仕様メモ、AI/Codexへの依頼文、実装レビュー用チェックリストに変換する材料を追加します。

Proをアンロック — $2.99

Proで得られるもの

  • UI判断メモ: 選んだパターン、却下した代替案、理由、リスク、検証計画をそのまま貼れる文面にします。
  • 拡張比較チェック: アクセシビリティリスク、モバイル適性、文量リスク、実装コスト、保守コスト、誤用注意を確認できます。
  • AI/Codex引き継ぎプロンプト: 役割、文脈、制約、受け入れ条件、フォールバック挙動まで含めた長めの依頼文を用意します。
  • 実装レビュー用チェックリスト: キーボード操作、フォーカス状態、reduced motion、空状態、エラー状態、ローディング、レスポンシブ確認を含みます。
  • 仕様書向け文面: GitHub Issue、デザインメモ、プロダクト仕様、小規模クライアント案件に貼れるMarkdown風テキストを用意します。

無料版とPro版の違い

  • 無料版: 50件のライブUIパターン例、検索、絞り込み、詳細パネル、短文プロンプト、お気に入り、履歴、2件比較。
  • Pro: パターンを選んだ後に必要になる判断・引き継ぎ・実装確認の材料。

Pro出力サンプル

判断: アカウント削除のような取り消せない操作には確認ダイアログを使う。明示的な意思確認が必要なため。

却下: トースト通知やインライン警告だけでは見落とされやすく、確認操作として不十分。

実装チェック: 明確なタイトル、影響範囲の説明、キャンセル優先、破壊的ボタンの視覚差、フォーカストラップ、Escape終了、スクリーンリーダー向けラベル。
