無料版でできること

無料版は、実装前に代表的なUIパターンを調べるための基本的なUIパターン辞典です。

  • 100 UI examples を見る。
  • UIパターン検索、カード一覧、詳細表示を使う。
  • ライブサンプルと Short AI prompt を使う。
  • お気に入りと最近見た項目を保存する。
  • 2件まで比較する。
  • Pro-only 50件を locked preview として見る。
  • Pro出力の一部Previewを見る。

Pro版でできること

Pro版は、選んだUIパターンをデザイン・プロダクト・実装に渡せる判断資料へ変えるUI実装判断・引き継ぎ資料生成ツールです。

  • 150 UI examples を見る。
  • 50 Pro-only samples の詳細を見る。
  • 最大5件比較を使う。
  • Pro-only sample preview / live sample を使う。
  • Decision Memo と Compare Memo を生成する。
  • Codex / GitHub Issue Prompt を生成する。
  • Acceptance Criteria、Accessibility Review、Mobile Review を生成する。
  • Full Markdown Export と JSON Export を使う。
  • Copy all / Copy prompt / Copy checklist / Copy memo を使う。

PreviewモードとPro解放済みの違い

Previewモード

このブラウザでは共通Proがまだ有効ではありません。

できること:

  • 無料100件を見る。
  • 2件まで比較する。
  • Pro-only 50件を locked preview で見る。
  • Pro出力の形式をPreviewとして見る。
  • 購入ボタンを押す。

できないこと:

  • 3〜5件比較。
  • Pro-only詳細表示。
  • Full Markdown Export。
  • JSON Export。
  • Copy all。

Pro解放済み

このブラウザでは共通Proが有効です。

できること:

  • 150件すべて見る。
  • Pro-only 50件の詳細を見る。
  • 最大5件比較する。
  • Decision Memo生成。
  • Compare Memo生成。
  • Codex / GitHub Issue Prompt生成。
  • Acceptance Criteria生成。
  • Accessibility / Mobile Review生成。
  • Full Markdown / JSON Export。
  • Copy all。

5-mode Pro Generator

Proでは、選択したUIパターンや比較結果を実装に渡せる資料へ変換する5モードを解放します。

モード価値
1. 判断メモ採用パターン、理由、却下案、リスク、検証計画を整理します。
2. 複数UI比較複数候補のトレードオフ、推奨、アクセシビリティ・モバイル・実装リスクを整理します。
3. Codex / GitHub Issue依頼文役割、文脈、制約、UI要件、状態、受け入れ条件を含む依頼文を作ります。
4. 受け入れ条件デスクトップ、モバイル、キーボード、フォーカス、読み込み、空、エラー、長文、reduced motionを確認できる条件を用意します。
5. Markdown一括出力Issue、仕様、実装メモに貼れるMarkdown文書としてまとめます。

購入後について

購入後、Stripeは /pro/unlock/?session_id={CHECKOUT_SESSION_ID} に戻ります。この共通解放ページがD1上の nicheworks_pro 権限を確認し、対応ツールで使うための補助状態をブラウザに保存します。

同じブラウザでは、タブやブラウザを閉じても通常Pro状態は維持されます。ただし、別ブラウザ・別端末・シークレットモード・サイトデータ削除後は再度解放が必要になる場合があります。localStorageは補助状態であり、D1 pro_entitlements がサーバー側の正です。
