UI Atlasとは

UI Atlasは、UIパターン名の辞書ではなく、選定判断まで支援する実用ツールです。ライブサンプル、適用場面、比較、短文プロンプトを一体化しています。

主に役立つ人

  • 限られた時間でUI判断を積み重ねる個人開発者・小規模チーム。
  • デザインと実装で「どれを選ぶか」を短時間で合意したい人。
  • AIに実装依頼する際、曖昧語ではなく具体語で指示したい人。

速くしたい意思決定

  • この操作に対して、どのパターンが安全か。
  • モバイル条件で避けるべき構成は何か。
  • 採用理由を短く言語化してAI依頼文に落とせるか。

ギャラリーやプロンプト集と違う点

  • 見栄え重視の一覧ではなく、適材適所の判断を中心に設計しています。
  • プロンプト単体ではなく、向く場面・避ける場面とセットで扱います。
  • 検索→詳細→比較→記録の一連導線を1つの作業面で回せます。

現在の提供範囲

無料版では、実用カタログ、ライブサンプル、詳細、2件比較、ローカルのお気に入り/履歴を提供しています。過剰なLP化はせず、ツール品質を優先します。
