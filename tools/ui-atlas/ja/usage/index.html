無料版の基本フロー

UI Atlasは「眺める辞典」ではなく「判断する辞典」です。候補を探し、使いどころを確認し、2件比較で迷いを減らして、最後に短文プロンプト化します。

1) 検索のコツ

  • 「ドロワーメニュー」「確認ダイアログ」「価格表」のような実務ワードで検索できます。
  • 名称だけでなく、別名・要約・用途・初心者向け説明も検索対象です。
  • 候補が多い場合は「モバイル」「チェックアウト」など文脈語を1つ追加すると絞りやすくなります。

2) フィルターで候補を整理

  • カテゴリで広い方向性を決める（ナビ、入力、情報表示など）。
  • 用途で操作意図を合わせる（確認系、短操作、情報探索など）。
  • モバイル適性 + 実装難易度を組み合わせると現実的な候補だけ残せます。

3) 詳細で判断精度を上げる

  • 各カードから詳細を開くと、拡大ライブサンプルを確認できます。
  • 「向いている場面 / 向いていない場面 / 実装メモ」を先に見て、誤用を避けてください。
  • 短文AIプロンプトはコピー後に自分の仕様語を追加して使う想定です。

4) 比較（無料版は2件まで）

  • カードから2件まで追加し、差分と選び方の目安を確認します。
  • 2件制限は、比較を早く終わらせるための意図的な上限です。
  • 1件以上選択中は、いつでもクリアできます。

5) ローカルお気に入り / 履歴

  • お気に入りはよく使うパターンをブラウザ内に保持します。
  • 最近見た項目は直前の検討ラインを戻るための補助です。
  • 現時点では端末間同期はありません（ローカル保存のみ）。

無料で使える範囲

検索、絞り込み、詳細、ライブサンプル、ローカルお気に入り/履歴、2件比較は現在すべて無料で利用できます。
