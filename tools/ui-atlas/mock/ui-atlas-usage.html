UI Atlas の使い方How to use UI Atlas

このページは、UI Atlas モックで何ができるか、何がまだモック段階かを短く説明するための usage ページです。This page explains what the UI Atlas mock already demonstrates and what is still placeholder-only.

広告枠（ad-top / 準備中）

1. できること1. What works already

  • 曖昧語を含む検索とサジェストFuzzy search with suggestions
  • カテゴリ・目的・デバイスでの絞り込みFiltering by category, purpose, and device fit
  • 一覧カードからの詳細表示Open detail from cards
  • 比較トレイへの追加と1対1比較Add to compare tray and 1-to-1 comparison
  • 短文AI依頼文のコピーCopy the short AI prompt

2. まだモック段階の部分2. Still mock-only

  • 保存済み比較セットSaved compare sets
  • 3件以上の比較3+ item comparison
  • 詳細AI依頼文の複数バリエーションMultiple full prompt variants
  • 実装メモの全文と高度な落とし穴一覧Full implementation notes and deeper pitfalls

3. 想定している使い方3. Intended workflow

まず曖昧な言い方で検索し、候補一覧で気になるUIを開き、右パネルで「向いている場面」「向いていない場面」「似たUIとの違い」を確認します。その後、比較トレイへ2件追加して見比べ、最後に短文AI依頼文をコピーする流れを想定しています。Start with a fuzzy phrase, open likely candidates, confirm the “best for”, “not for”, and “differences” sections, then add two items to the compare tray and copy the short AI prompt.

このツールが役に立ったら、開発継続のためのご支援をいただけると嬉しいです。If this tool seems useful, support helps keep development going.
