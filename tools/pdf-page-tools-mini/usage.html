PDFページ操作ミニ｜使い方 PDF Page Tools Mini — Usage

結合・抽出・並べ替え・回転を、アップロードなしで。 Merge, extract, reorder, rotate — no upload.

このページについて About this page

PDFページ操作ミニの基本手順と、対応範囲・エラー時の対処をまとめています。 This page explains how to use PDF Page Tools Mini, supported features, and troubleshooting.

※このツールはブラウザ内で処理します（サーバーにアップロードしません）。 *This tool processes PDFs locally in your browser (no upload).

できること（5機能） What you can do (5 features)

  • 結合（Merge）：複数PDFを1つにMerge: combine multiple PDFs into one
  • 抽出（Extract）：指定範囲だけを新PDFで保存Extract: save selected page ranges as a new PDF
  • 削除（Delete）：不要ページを除外して保存Delete: remove pages and save
  • 並べ替え（Reorder）：ページ順を入れ替えて保存Reorder: rearrange pages and save
  • 回転（Rotate）：ページごとに90°単位で回転して保存Rotate: rotate pages by 90° increments and save

基本手順（追加 → 編集 → 保存） Basic workflow (Add → Edit → Download)

  1. PDFを追加Add PDFs

    「PDFを追加」から読み込みます。複数選択やドラッグ&ドロップも可能です。 Use “Add PDFs” to load files. Multiple selection and drag & drop are supported.

  2. ページを編集Edit pages

    ページカードで「回転」「削除」「↑↓（並べ替え）」を行います。PCではドラッグでの並べ替えも利用できます。 Use page cards to rotate, delete, and move pages up/down. On desktop, drag & drop reorder may be available.

  3. 保存してダウンロードDownload

    「保存してダウンロード」で現在の状態をPDFとして出力します（回転も反映されます）。 “Download” exports the current state as a PDF (rotations are applied).

抽出（範囲指定）の書き方 Extract (range syntax)

ページ範囲欄に、次の形式で入力します： Enter ranges using this format: 

1-3,5,8-10
  • カンマ区切りで複数指定できますUse commas to specify multiple ranges
  • 数字は「今見えているページ番号（#1,#2…）」ですNumbers refer to the current visible page order (#1, #2…)

対応しないこと（非対応） Not supported

  • 圧縮（Compress）、OCR、注釈、テキスト編集Compression, OCR, annotations, text editing
  • パスワード解除、署名、画像化、透かしPassword removal, signing, image conversion, watermark

※パスワード付きPDFは非対応（解除は行いません）。 *Encrypted PDFs are not supported (no password removal).

エラーと対処 Errors & troubleshooting

PDFとして読み込めないCannot read as PDF

別のPDFを試してください。拡張子が.pdfでも中身がPDFでない場合があります。 Try another PDF. Some files may not be valid PDFs even if they end with .pdf.

処理に失敗する（重い/落ちる）Processing fails (heavy / crashes)

端末のメモリ不足の可能性があります。ページ数を減らす・分割する・別ブラウザで試してください。 This may be a memory issue. Try fewer pages, split the PDF, or use another browser.

抽出の範囲が通らないInvalid extract range

例の形式（1-3,5,8-10）で入力してください。範囲は現在のページ数以内にしてください。 Use the example format (1-3,5,8-10) and keep numbers within the current page count.

出力するページがないNo pages to export

削除をUndoで戻すか、PDFを追加してください。 Undo deletion or add PDFs.

出力ファイル（命名） Output filenames

  • 単一PDF：元名_edited.pdf / 元名_extracted.pdfSingle PDF: original_edited.pdf / original_extracted.pdf
  • 複数PDF：pdf-page-tools-mini_edited_YYYYMMDD_HHMM.pdf などMultiple PDFs: pdf-page-tools-mini_edited_YYYYMMDD_HHMM.pdf etc.

免責（端末依存） Disclaimer (device/browser dependent)

このツールは端末内（ブラウザ）で処理します。PDFのサイズや端末性能により、動作が不安定になる場合があります。 This tool runs locally in your browser. Performance may vary depending on PDF size and device capabilities.
