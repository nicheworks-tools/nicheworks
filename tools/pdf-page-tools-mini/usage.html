PDFをアップロードせずに、ページ操作を行うための手順と注意点です。

How to edit PDF pages without uploading your files.

このツールでできること

What this tool can do

  • 複数PDFの結合
  • Merge multiple PDFs.
  • 指定ページ範囲の抽出
  • Extract selected page ranges.
  • ページ削除、並べ替え、90度単位の回転
  • Delete, reorder, and rotate pages by 90-degree increments.
  • 編集後PDFの別ファイル保存
  • Save the edited result as a separate PDF file.

処理はブラウザ内で完結します。元のPDFは変更されません。

Processing stays in your browser. Original PDFs are not changed.

基本手順

Basic workflow

  1. PDFを追加：ボタン、複数選択、ドラッグ&ドロップでPDFを読み込みます。
  2. Add PDFs: Use the button, multi-select, or drag and drop.
  3. ページを編集：ページカードで削除、回転、上下移動を行います。PCではドラッグ並べ替えも使えます。
  4. Edit pages: Use page cards to delete, rotate, or move pages. Desktop drag and drop may also be available.
  5. 保存：「保存してダウンロード」で現在のページ順と回転を反映したPDFを保存します。
  6. Download: Save the current order and rotations as a PDF.

抽出範囲の書き方

Extract range syntax

ページ範囲欄には、現在表示されているページ番号を次の形式で入力します。

Use the current visible page numbers and enter ranges like this.

1-3,5,8-10
  • カンマで複数指定できます。
  • Use commas for multiple ranges.
  • 削除や並べ替え後は、画面上の #1, #2 ... の番号を基準にします。
  • After deleting or reordering, use the visible #1, #2 ... order.

対応しないこと

Not supported

  • PDF圧縮、OCR、注釈編集、本文テキスト編集
  • PDF compression, OCR, annotation editing, or text editing.
  • パスワード付きPDFの解除や回避
  • Unlocking or bypassing password-protected PDFs.
  • 電子署名、透かし、PDFの画像化
  • Digital signing, watermarking, or converting PDFs to images.

重いPDF・特殊PDFで失敗する場合

If a heavy or special PDF fails

画像が多いPDF、100ページ以上のPDF、数十MB以上のPDF、破損PDF、一部の特殊PDFは端末によって失敗することがあります。

Image-heavy PDFs, PDFs over 100 pages, PDFs of several dozen MB, corrupted PDFs, or some special PDFs may fail depending on your device.

  • ページ数を減らす
  • Reduce the page count.
  • 先に小さなPDFへ分割する
  • Split the PDF into smaller files first.
  • PCブラウザや別ブラウザで試す
  • Try a desktop browser or another browser.
