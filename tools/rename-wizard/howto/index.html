Rename Wizard：使い方

「アップロードで弾かれるファイル名」を、まとめて安全な名前に整形。

広告枠（準備中）

このツールでできること

  • 一括複数ファイルの名前ルールをまとめて整形
  • 変換全角→半角、空白→_、連続空白の整理
  • 除去禁止文字（\ / : * ? " < > | など）を除去/置換
  • 安全処理はローカルで完結（サーバー送信なし）
ツールを開く English guide

よくある症状

  • ファイルアップロードで「無効なファイル名」と言われる
  • WindowsとMacで文字化け/表示崩れが起きる
  • スペースや全角記号が混ざって、並びや検索がしづらい
  • 連番や命名ルールがバラバラで、管理が破綻する

関連キーワード：ファイル名 一括 変更 / 禁止文字 置換 / 全角 半角 ファイル名 / 空白 アンダースコア / rename tool

使い方（最短3ステップ）

  1. 整形したいファイル名（または一覧）をツールに入力/取り込む
  2. 適用したいルール（空白→_、禁止文字除去など）をONにする
  3. 結果をコピーしてリネームに使う（または出力を使って一括適用）

コツ：最初は「禁止文字除去＋空白置換＋全角→半角」だけで9割の事故を潰せる。

注意（できないこと・限界）

  • 元のファイル自体は変更しません（名前ルールの整形/提案です）。
  • サービス側の独自制約（最大文字数など）は別途あります。
  • 同名衝突が起きる場合は、連番付与などで回避してください。
