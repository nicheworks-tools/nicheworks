使い方

引っ越しやることチェックリストの使い方と注意点。

ツールに戻る

1. このツールでできること

引っ越し日・家族構成・住居タイプから、30日前〜引越し後の作業を時系列で整理して表示します。チェックリスト生成とチェック状態の保存処理はブラウザ内で行います。

2. 使い方の手順

  1. 引っ越し日を選びます。
  2. 単身/家族、賃貸/持ち家を選びます。
  3. 必要なら印刷用メモを入力します。住所・連絡先・氏名・部屋番号などは入れないでください。
  4. 「チェックリストを生成」を押します。
  5. 終わった項目にチェックを入れます。チェック状態と条件識別用の引っ越し日・家族構成・住居タイプが、このブラウザのlocalStorageに保存されます。
  6. 必要なら「印刷 / PDF保存」を押し、ブラウザの印刷画面からPDFとして保存します。

3. 対応 / 非対応

  • 対応：チェック保存（localStorage）、進捗表示、印刷/PDF保存、JP/EN切替
  • 保存されるもの：チェック状態、条件識別用の引っ越し日・家族構成・住居タイプ
  • 保存されないもの：印刷用メモ
  • 非対応：クラウド同期、共有リンク、住所などの保存、個別事情の完全反映、契約・自治体期限の保証

4. エラーについて

引っ越し日が未入力の場合、生成できません。全保存データ削除後も、日付が未入力の場合は不要な再生成を行わず、空の状態を維持します。

5. 出力形式

30日前、14日前、7日前、5日前、3日前、前日、当日、引越し後のブロックで表示します。印刷用メモは保存されませんが、印刷やPDFには表示される場合があります。

6. FAQ

チェック状態と条件識別用の設定は、このブラウザのlocalStorageに保存されます。別端末・別ブラウザには引き継がれず、ブラウザデータを削除すると消えます。ページ表示のため、GA4 / AdSense / Cloudflare Analytics などの広告・解析タグが読み込まれる場合があります。

7. 免責

本ツールは一般的な参考リストです。退去通知期限、原状回復、違約金、ライフライン停止/開始、自治体手続きの期限は、契約書・管理会社・自治体・各サービスの公式情報を優先してください。
