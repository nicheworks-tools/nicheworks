Pattern Atlas の使い方

模様の探し方、色編集、素材出力、文化的注意についての説明ページです。

English version

このツールでできること

Pattern Atlasは、世界の伝統文様・民族文様・地域模様を、意味や用途から探し、色を編集し、SVG / PNG / CSS素材として書き出すためのツールです。

基本の流れ

  1. 検索またはフィルタで模様を探します。
  2. 意味・地域・用途・注意点を確認します。
  3. 色を変更し、タイルや用途別プレビューで見た目を確認します。
  4. SVG / PNG / CSS形式で書き出します。

出力形式

SVG は拡大しても劣化しにくいベクター形式です。

PNG はWebや資料に使いやすい一般的な画像形式です。

CSS はWeb背景として使うためのコード出力です。

文化的注意

このツールの模様は、伝統・地域・民族・宗教的背景を持つ文様を参考にする場合があります。出力素材は公式な歴史復元や文化的認証素材ではありません。商用利用や公的利用では、必要に応じて追加調査を行ってください。

保存されるデータ

MVPでは、言語設定やお気に入りなどの軽い設定をブラウザ内のlocalStorageに保存する可能性があります。個人情報やアップロード画像をサーバーへ送る設計にはしません。

FAQ

出力した素材は自由に使えますか？
模様によって文化的背景や商用利用上の注意が異なります。各詳細の注意文を確認してください。

このサイトの模様は歴史的に正確な復元ですか？
いいえ。原則として、伝統文様等に着想を得た現代的な再構成として扱います。

入力内容はサーバーに送られますか？
MVPはブラウザ内で動く静的ツールとして設計します。
