# Pattern Atlas AGENTS

Scope: `tools/pattern-atlas/` only.

## Required

- Keep all Pattern Atlas files inside `tools/pattern-atlas/`.
- Use NicheWorks-native, restrained UI styling.
- Use static HTML, CSS, and vanilla JavaScript for MVP.
- Use relative paths inside this directory.
- Prefix tool-specific CSS classes with `pa-`.
- Scope JavaScript DOM access to the Pattern Atlas root element.

## Forbidden

- Do not add Pattern Atlas-specific files directly under `/tools/`.
- Do not edit other tool directories for Pattern Atlas work.
- Do not use the UI Atlas-style permanent desktop 3-column layout.
- Do not add React, Vue, Next.js, or external API dependencies for MVP.
- Do not store large pre-rendered image sets.
- Do not skip context and usage warnings for traditional, religious, ceremonial, or culturally sensitive patterns.
