このページの位置づけ

全単位一覧ページの補足として、特に問い合わせや混同が起きやすい単位を詳しく説明します。換算値はUnitMaster内の代表値です。専門用途では公式規格・専門資料・測定条件を確認してください。

坪

換算値：1坪 = 3.305785㎡

土地・建物・部屋の広さを表す日本の伝統的な面積単位です。長さではなく面積の単位なので、UnitMasterでは面積カテゴリだけで扱います。

主な使われ方：土地面積、住宅・部屋・店舗の広さ、不動産広告、建築関連の説明。

混同注意：「10坪」は長さではなく、面積として約33.05785㎡を意味します。

例：6坪 = 約19.83471㎡、30㎡ = 約9.075坪

尺

換算値：1尺 = 約0.303m

日本の伝統的な長さ単位です。寸、分、間などの関係を理解する基準になり、建築・和裁・木工・古い文献などで見かけます。

関係：1尺 = 10寸。1寸 = 10分。1間 = 6尺として扱われることがあります。

注意：時代・地域・用途で扱いが異なる場合があります。

例：2尺 = 約0.606m、6尺 = 約1.818m

寸

換算値：1寸 = 約0.0303m

尺の10分の1にあたる長さ単位です。和裁、木工、伝統工芸、古い寸法表現などで使われます。

関係：1尺 = 10寸。1寸 = 10分。

例：5寸 = 約15.15cm

間

換算値：1間 = 約1.818m

建築や和室、柱間、古い住宅寸法などで使われてきた長さ単位です。尺との関係で理解されることが多く、1間 = 6尺として扱われます。

主な使われ方：建築寸法、部屋の表現、古い図面、和室や畳まわりの説明。

混同注意：畳数や坪数とは別の「長さ」の単位です。面積を表す場合は坪や㎡を使います。

例：2間 = 約3.636m

里

換算値：1里 = 約3927m

道のりや古い地理表現で使われてきた長さ単位です。現代の日常計測ではあまり使われませんが、古文・歴史資料・地名や道のりの説明で見かけます。

主な使われ方：歴史資料、古い旅程、地理・道のりの表現。

混同注意：時代や制度により扱いが変わる場合があります。UnitMasterでは代表的な近似値として扱います。

例：2里 = 約7.854km

匁

換算値：1匁 = 3.75g

日本の伝統的な重さ単位です。真珠、貴金属、薬種、古い商取引表現などで使われてきました。

注意：業界によって慣用的な使われ方が残るため、実務では業界ごとの表記や規格を確認してください。

例：10匁 = 37.5g

斤

換算値：1斤 = 600g

尺貫法の重さ単位です。食品や古い重量表現で使われます。

混同注意：現代の食パンなどで使われる「1斤」は商品上の呼び方で、UnitMasterの換算値600gと一致しない場合があります。

例：2斤 = 1200g

合

換算値：1合 = 0.18039L

米や酒などで使われる日本の伝統的な体積単位です。日常では「米1合」のような形で今もよく使われます。

関係：10合 = 1升。体積の目安であり、炊飯後の量や重量とは別です。

例：5合 = 約0.90195L

升

換算値：1升 = 1.8039L

10合にあたる体積単位です。米、酒、枡、古い容量表現などで使われます。

関係：1升 = 10合。10升 = 1斗。

例：2升 = 約3.6078L

cup / カップ

換算値：UnitMasterでは 1cup = 0.24L として扱います。

料理やレシピで使われる体積単位です。ただし、国・地域・レシピの体系によって1カップの量が異なる場合があります。

主な使われ方：海外レシピ、料理、飲料、粉類や液体の目安。

混同注意：日本の計量カップ、米用カップ、米国カップなどは同じ「cup」でも値が違う場合があります。正確な料理・製造用途ではレシピ側の定義を確認してください。

例：2cup = 0.48L（UnitMaster上の代表値）

畝

換算値：1畝 = 99.1736㎡

農地などで使われる日本の伝統的な面積単位です。現代では日常的な面積表現としては少なめですが、農地や古い土地資料で見かけることがあります。

関係：10畝 = 1反として扱われます。

混同注意：土地面積の単位であり、長さではありません。

例：5畝 = 約495.868㎡

反

換算値：1反 = 991.736㎡

農地面積で使われる伝統的な単位です。田畑の広さや農業関連の資料で見かけます。

関係：1反 = 10畝、10反 = 1町として扱われます。

混同注意：不動産広告でよく見る坪とは別系統の面積単位です。

例：2反 = 約1983.472㎡

町

換算値：1町 = 9917.36㎡

大きな土地・農地面積を表す伝統的な単位です。現代の一般的な住宅面積よりも大きなスケールの土地で使われます。

関係：1町 = 10反として扱われます。

混同注意：町名や行政区画の「町」とは別です。UnitMasterでは面積単位として扱います。

例：1町 = 約0.991736ha

psi

換算値：1psi = 6894.76Pa

pounds per square inch の略で、1平方インチあたりのポンド力を表す圧力単位です。英米圏の機器やタイヤ空気圧でよく使われます。

注意：安全に関わる圧力は必ず機器の取扱説明書や公式仕様を確認してください。

例：30psi = 約206842.8Pa = 約206.843kPa

bar / バール

換算値：1bar = 100000Pa

工業、気象、機器表示、圧力計などで見かける圧力単位です。SI単位のPaと合わせて使われることがあります。

主な使われ方：機械、圧力計、コンプレッサー、気象・工業関連の資料。

混同注意：bar、hPa、atm、psi は用途が違います。機器の安全や規格に関わる場合は公式仕様を確認してください。

例：1bar = 1000hPa

torr / トル

換算値：1torr = 133.322Pa

真空や圧力表現で使われる単位です。真空装置、実験、古い圧力表記などで見かけることがあります。

主な使われ方：真空、実験装置、圧力資料。

混同注意：真空や研究用途では条件・測定方法・単位系が重要です。換算値だけで仕様判断しないでください。

例：760torr ≒ 1atm

knot / ノット

換算値：1knot = 0.514444m/s

船舶・航空で使われる速度単位です。海上や航空の速度、風速表現などで見かけます。

例：10knot = 約5.14444m/s = 約18.52km/h

inch / インチ

換算値：1inch = 2.54cm

海外製品、画面サイズ、工具、配管、ネジ・部品寸法などでよく使われます。

注意：画面サイズの「インチ」は対角線の長さを表すことが多く、幅や高さそのものではありません。

例：15inch = 38.1cm

mile / マイル

換算値：1mile = 1609.344m

英米圏の道路距離、移動距離、速度のmphとセットでよく使われます。

関係：1mile = 約1.609344km。mph は miles per hour の略です。

例：5mile = 約8.04672km

parsec / パーセク

換算値：1parsec = 約3.0857e16m

天文学で使われる巨大な距離単位です。恒星間距離や銀河スケールの距離を表すときに使われます。

主な使われ方：天文学、宇宙距離、恒星・銀河関連の資料。

混同注意：日常的な距離ではなく、宇宙規模の距離表現です。kmやmileと同じ感覚で扱うと桁が大きくずれます。

例：1parsec ≒ 3.26光年

lightyear / 光年

換算値：1lightyear = 約9.4607e15m

光が1年で進む距離を基準にした長さ単位です。名前に「年」が入っていますが、時間ではなく距離の単位です。

主な使われ方：宇宙距離、恒星までの距離、天文学の説明。

混同注意：光年は時間単位ではありません。「何年かかるか」ではなく「どれくらい離れているか」を表します。

例：1光年 = 約9.4607兆km
