坪

換算値：1坪 = 3.305785㎡

土地・建物・部屋の広さを表す日本の伝統的な面積単位です。長さではなく面積の単位なので、UnitMasterでは面積カテゴリだけで扱います。

主な使われ方：土地面積、住宅・部屋・店舗の広さ、不動産広告、建築関連の説明。

混同注意：「10坪」は長さではなく、面積として約33.05785㎡を意味します。

例：6坪 = 約19.83471㎡、30㎡ = 約9.075坪

尺

換算値：1尺 = 約0.303m

日本の伝統的な長さ単位です。寸、分、間などの関係を理解する基準になり、建築・和裁・木工・古い文献などで見かけます。

関係：1尺 = 10寸。1寸 = 10分。1間 = 6尺として扱われることがあります。

注意：時代・地域・用途で扱いが異なる場合があります。

例：2尺 = 約0.606m、6尺 = 約1.818m

寸

換算値：1寸 = 約0.0303m

尺の10分の1にあたる長さ単位です。和裁、木工、伝統工芸、古い寸法表現などで使われます。

関係：1尺 = 10寸。1寸 = 10分。

例：5寸 = 約15.15cm

匁

換算値：1匁 = 3.75g

日本の伝統的な重さ単位です。真珠、貴金属、薬種、古い商取引表現などで使われてきました。

注意：業界によって慣用的な使われ方が残るため、実務では業界ごとの表記や規格を確認してください。

例：10匁 = 37.5g

斤

換算値：1斤 = 600g

尺貫法の重さ単位です。食品や古い重量表現で使われます。

混同注意：現代の食パンなどで使われる「1斤」は商品上の呼び方で、UnitMasterの換算値600gと一致しない場合があります。

例：2斤 = 1200g

合

換算値：1合 = 0.18039L

米や酒などで使われる日本の伝統的な体積単位です。日常では「米1合」のような形で今もよく使われます。

関係：10合 = 1升。体積の目安であり、炊飯後の量や重量とは別です。

例：5合 = 約0.90195L

升

換算値：1升 = 1.8039L

10合にあたる体積単位です。米、酒、枡、古い容量表現などで使われます。

関係：1升 = 10合。10升 = 1斗。

例：2升 = 約3.6078L

psi

換算値：1psi = 6894.76Pa

pounds per square inch の略で、1平方インチあたりのポンド力を表す圧力単位です。英米圏の機器やタイヤ空気圧でよく使われます。

注意：安全に関わる圧力は必ず機器の取扱説明書や公式仕様を確認してください。

例：30psi = 約206842.8Pa = 約206.843kPa

knot / ノット

換算値：1knot = 0.514444m/s

船舶・航空で使われる速度単位です。海上や航空の速度、風速表現などで見かけます。

例：10knot = 約5.14444m/s = 約18.52km/h

inch / インチ

換算値：1inch = 2.54cm

海外製品、画面サイズ、工具、配管、ネジ・部品寸法などでよく使われます。

注意：画面サイズの「インチ」は対角線の長さを表すことが多く、幅や高さそのものではありません。

例：15inch = 38.1cm

mile / マイル

換算値：1mile = 1609.344m

英米圏の道路距離、移動距離、速度のmphとセットでよく使われます。

関係：1mile = 約1.609344km。mph は miles per hour の略です。

例：5mile = 約8.04672km
