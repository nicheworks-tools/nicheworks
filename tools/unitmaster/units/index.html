このページで分かること

変換フォームに出てくる単位が何を表すのか、どの場面で使われるのか、UnitMasterではどの換算値で扱うのかを確認できます。伝統単位は値が揺れやすいため、意味・関係・注意点を通常単位より厚めに記載しています。

まず読む注意

  • 換算値はUnitMaster内部の代表的な換算係数に合わせた目安です。
  • 伝統単位は時代・地域・用途により値が異なる場合があります。
  • 医療、建築、工業設計、契約、取引、研究、法規制では、公式規格・専門資料・測定条件を確認してください。

長さの単位

mm / ミリメートル

換算値：0.001m

1メートルの1000分の1を表す長さの単位です。細かい部品、厚み、製図寸法などで使われます。

関係：10mm = 1cm、1000mm = 1m。

cm / センチメートル

換算値：0.01m

1メートルの100分の1を表す単位です。身長、日用品、短い距離の説明でよく使われます。

関係：100cm = 1m。

m / メートル

換算値：1m

国際単位系における基本的な長さの単位です。距離、寸法、高さ、幅などの基準になります。

km / キロメートル

換算値：1000m

1000メートルを表す長距離向けの単位です。道路距離、移動距離、地図上の距離に使われます。

inch / インチ

換算値：0.0254m

ヤード・ポンド法の長さ単位です。画面サイズ、工具、配管、海外製品寸法などで見かけます。

関係：12inch = 1ft。

ft / フィート

換算値：0.3048m

12インチに相当する長さ単位です。身長、航空高度、建築寸法などで使われます。

yard / ヤード

換算値：0.9144m

3フィートに相当する単位です。スポーツ、布地、距離表現で使われます。

mile / マイル

換算値：1609.344m

ヤード・ポンド法の長距離単位です。英米圏の道路距離、移動距離、速度表記と一緒に使われます。

寸

換算値：約0.0303m

尺の10分の1にあたる日本の伝統的な長さ単位です。和裁、木工、古い寸法表現などで使われます。

関係：1尺 = 10寸、1寸 = 10分。

注意：時代・地域・用途により値が異なる場合があります。

尺

換算値：約0.303m

日本の伝統的な長さ単位で、寸・分・間などの基準にもなります。建築、和裁、古典文献、伝統工芸の寸法表現で使われます。

関係：1尺 = 10寸。1間 = 6尺として扱われることがあります。

注意：UnitMasterでは代表的な近似値として扱います。

分

換算値：約0.00303m

寸の10分の1にあたる小さな長さ単位です。細かな寸法、伝統的な加工・計測表現で使われます。

間

換算値：約1.818m

建築や部屋の寸法で使われてきた長さ単位です。和室、畳、建築寸法、間取りなどと関係します。

関係：1間 = 6尺として扱われることがあります。

里

換算値：約3927m

道のりや距離を表す日本の伝統的な単位です。古い地理表現、道のり、歴史資料などで見かけます。

furlong / ハロン

換算値：201.168m

英米系の伝統的な距離単位です。競馬や古い距離表現で使われます。

chain / チェーン

換算値：20.1168m

測量で使われてきた長さ単位です。土地測量や古い測地資料で見かけます。

league / リーグ

換算値：4828.032m

長距離を表す古典的な単位です。航海、古い距離表現、文学表現で使われます。

angstrom / オングストローム

換算値：1e-10m

非常に小さい長さを表す単位です。原子・分子レベルの長さ表現で使われます。

micrometer / マイクロメートル

換算値：1e-6m

100万分の1メートルを表す単位です。微細加工、粒子、細胞サイズなどで使われます。

parsec / パーセク

換算値：約3.0857e16m

天文学で使われる距離単位です。恒星間距離など、宇宙規模の距離表現で使われます。

light-year / 光年

換算値：約9.4607e15m

光が1年間に進む距離を表す単位です。宇宙規模の距離を説明するときに使われます。

重さの単位

g / グラム

換算値：1g

質量を表す基本的な単位のひとつです。食品、日用品、少量の重さなどで使われます。

kg / キログラム

換算値：1000g

国際単位系の質量の基本単位です。体重、荷物、食品、工業製品など広く使われます。

lb / ポンド

換算値：453.59237g

ヤード・ポンド法の重さ単位です。英米圏の体重・食品・荷物重量などで使われます。

oz / オンス

換算値：28.3495231g

ポンドの16分の1にあたる重さ単位です。食品、飲料、軽量物の表記で使われます。

匁

換算値：3.75g

日本の伝統的な重さ単位です。真珠、貴金属、薬種、古い商取引表現で使われてきました。

注意：用途によって実務上の扱いが異なる場合があります。

斤

換算値：600g

尺貫法の重さ単位です。食品や古い重量表現で使われます。

注意：現代の食パンなどの商品上の「斤」と換算単位が一致しない場合があります。

貫

換算値：3750g

尺貫法における大きな重さ単位です。古い商取引、体重、荷重の歴史的表現で見かけます。

dram / ドラム

換算値：1.771845g

ヤード・ポンド法系の小さな重さ単位です。古い薬用・計量表現で見かける場合があります。

grain / グレイン

換算値：0.06479891g

非常に小さな重さを表す単位です。専門用途では公式定義や規格を確認してください。

温度の単位

C / 摂氏

意味：水の氷点を0度、沸点を100度とする温度目盛です。

使われ方：日本を含む多くの国の日常温度、気象、料理、機器表示。

F / 華氏

意味：主に米国などで使われる温度目盛です。

使われ方：米国の気温、料理、機器表示。

K / ケルビン

意味：絶対零度を0Kとする温度単位です。

使われ方：科学、物理、研究用途。

注意：0K未満はUnitMasterではエラー扱いです。

体積の単位

ml / ミリリットル

換算値：0.001L

1リットルの1000分の1を表す体積単位です。飲料、薬液、料理などで使われます。

L / リットル

換算値：1L

液体や容積を表す一般的な単位です。飲料、燃料、容器容量などで使われます。

cup / カップ

換算値：0.24L

料理などで使われる体積単位です。国やレシピにより値が異なる場合があります。

合

換算値：0.18039L

米や酒などで使われる日本の伝統的な体積単位です。

関係：10合 = 1升。

升

換算値：1.8039L

10合にあたる体積単位です。米、酒、枡、古い容量表現で使われます。

関係：10升 = 1斗。

斗

換算値：18.039L

10升にあたる大きな体積単位です。農産物や古い商取引・容量表現で使われます。

面積の単位

mm²

換算値：1e-6m²

非常に小さな面積を表します。部品寸法や細かな面積表現に使われます。

cm²

換算値：1e-4m²

小さな面積を表します。図形、紙面、身近な面積の説明に使われます。

m² / 平方メートル

換算値：1m²

面積を表す基本的な単位です。部屋、土地、建物、床面積などで使われます。

km² / 平方キロメートル

換算値：1,000,000m²

広い地域や土地面積を表します。市区町村、森林、湖、国土面積などで使われます。

坪

換算値：3.305785m²

土地・建物・部屋の広さで使われる日本の伝統的な面積単位です。

注意：坪は長さではなく面積の単位です。

畝

換算値：99.1736m²

農地などで使われる伝統的な面積単位です。

反

換算値：991.736m²

農地面積で使われる単位です。

関係：10畝 = 1反として扱われることがあります。

町

換算値：9917.36m²

大きな土地・農地面積を表す伝統的な単位です。

関係：10反 = 1町として扱われることがあります。

m/s

km/h

換算値：0.277778m/s

1時間あたりに進むキロメートル数を表します。車、道路、移動速度で使われます。

mph

1時間あたりに進むマイル数を表します。英米圏の道路速度でよく使われます。

knot / ノット

league/h

リーグを1時間あたりの距離として扱う古典的な速度表現です。実務用途では標準的な速度単位を確認してください。

圧力の単位

Pa / パスカル

換算値：1Pa

国際単位系の圧力単位です。科学、工学、計測で使われます。

hPa / ヘクトパスカル

換算値：100Pa

気圧や天気予報でよく使われる単位です。

bar / バール

換算値：100000Pa

圧力を表す単位で、工業・気象・機器表示などで見かけます。

atm / 標準気圧

換算値：101325Pa

標準大気圧を基準にした単位です。

torr / トル

換算値：133.322Pa

真空や圧力表現で使われる単位です。専門用途では測定条件を確認してください。

psi

換算値：6894.76Pa

1平方インチあたりのポンド力を表す圧力単位です。タイヤ空気圧、機械、海外製品の圧力表示で使われます。

伝統単位まとめ

UnitMasterで採用している伝統単位は、長さ系の寸・尺・分・間・里、重さ系の匁・斤・貫、体積系の合・升・斗、面積系の坪・畝・反・町です。いずれも代表的な近似値として扱います。

  • 長さ系：1尺 = 10寸、1寸 = 10分、1間 = 6尺として扱われることがあります。
  • 体積系：10合 = 1升、10升 = 1斗として扱われることがあります。
  • 面積系：坪は面積単位です。長さ単位ではありません。
  • 重さ系：斤は商品表記と換算単位が一致しない場合があります。

よくある混同

  • 坪：長さではなく面積の単位です。土地や建物の広さで使います。
  • カップ：国やレシピによって量が異なります。このページではUnitMaster内の値として扱います。
  • 斤：商品名・慣用表現の「斤」と、換算単位としての斤が一致しない場合があります。
  • psi / torr：専門的な圧力用途では測定条件と公式規格を確認してください。
