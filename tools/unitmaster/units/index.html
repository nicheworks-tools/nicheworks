まず読む注意

  • 換算値はUnitMaster内部の代表的な換算係数に基づく目安です。
  • 伝統単位は時代・地域・用途により値が異なる場合があります。
  • 医療、建築、工業設計、契約、取引、研究、法規制では公式規格・専門資料・測定条件を確認してください。

このページの役割

このページは、変換フォームに出てくる単位の意味をざっくり理解するための解説ハブです。すべてのカードを長く並べるのではなく、カテゴリごとの考え方と、混同しやすい単位の入口を整理しています。

長さ

距離・寸法・高さ・幅を表す単位です。

mmcmmkminchftmile寸尺間

画面サイズのinch、建築・和裁で出る尺・寸、天文学の光年やパーセクは用途が大きく異なります。

重さ

食品、体重、荷物、貴金属などの重さを表します。

gkglboz匁斤貫

斤は商品名としての使われ方と、換算単位としての使われ方がズレる場合があります。

温度

日常温度、気象、料理、科学用途で使われます。

C / 摂氏F / 華氏K / ケルビン

Kは絶対零度を基準にするため、0K未満は扱いません。

体積

液体・容器・米・酒・料理などの量を表します。

mlLcup合升斗

cupは国やレシピで値が変わる場合があります。合・升は米や酒でよく見かけます。

面積

土地、部屋、建物、農地などの広さを表します。

m²km²坪畝反町

坪は長さではなく面積です。部屋や土地の広さでよく使われます。

速度・圧力

移動速度、風速、気圧、タイヤ空気圧、機器表示などで使われます。

m/skm/hmphknotPahPabarpsi

psiは安全に関わる機器やタイヤ空気圧で使われるため、公式仕様を確認してください。

伝統単位について

寸・尺・分・間・里、匁・斤・貫、合・升・斗、坪・畝・反・町などは、時代・地域・用途により扱いが異なる場合があります。UnitMasterでは代表的な近似値として扱います。
