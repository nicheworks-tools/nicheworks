換算値について

  • このページの換算値は、UnitMaster内部の代表的な換算係数に合わせた目安です。
  • 伝統単位は時代・地域・用途により値が異なる場合があります。
  • 医療、建築、工業設計、契約、取引、研究、法規制では、公式規格・専門資料・測定条件を確認してください。

長さの単位

mm / ミリメートル

換算値：0.001m

1メートルの1000分の1を表す長さの単位です。

使われ方：部品、厚み、寸法、製図。

cm / センチメートル

換算値：0.01m

1メートルの100分の1を表す長さの単位です。

使われ方：身長、日用品、短い距離。

m / メートル

換算値：1m

国際単位系の基本的な長さの単位です。

km / キロメートル

換算値：1000m

1000メートルを表す長距離向けの単位です。

inch / インチ

換算値：0.0254m

ヤード・ポンド法の長さ単位です。

ft / フィート

換算値：0.3048m

12インチに相当する長さ単位です。

yard / ヤード

換算値：0.9144m

3フィートに相当する長さ単位です。

mile / マイル

換算値：1609.344m

ヤード・ポンド法の長距離単位です。

寸

換算値：約0.0303m

尺の10分の1にあたる日本の伝統的な長さ単位です。

使われ方：和裁、木工、古い寸法表現。

注意：時代・地域・用途により値が異なる場合があります。

尺

換算値：約0.303m

日本の伝統的な長さ単位で、寸や間などの基準にもなります。

使われ方：建築、和裁、古典文献、伝統的な寸法表現。

分

換算値：約0.00303m

寸の10分の1にあたる小さな長さ単位です。

間

換算値：約1.818m

建築や部屋の寸法で使われてきた長さ単位です。

使われ方：和室、畳、建築寸法、間取り。

里

換算値：約3927m

道のりや距離を表す日本の伝統的な単位です。

furlong / ハロン

換算値：201.168m

英米系の伝統的な距離単位です。

chain / チェーン

換算値：20.1168m

測量で使われてきた長さ単位です。

league / リーグ

換算値：4828.032m

長距離を表す古典的な単位です。

angstrom / オングストローム

換算値：1e-10m

原子・分子レベルの長さ表現で使われます。

micrometer / マイクロメートル

換算値：1e-6m

微細加工、粒子、細胞サイズなどで使われます。

parsec / パーセク

換算値：約3.0857e16m

天文学で使われる距離単位です。

light-year / 光年

換算値：約9.4607e15m

光が1年間に進む距離を表す単位です。

kg / キログラム

温度の単位

K / ケルビン

km² / 平方キロメートル

町

伝統単位まとめ

UnitMasterで採用している伝統単位は、寸・尺・分・間・里・匁・斤・貫・合・升・斗・坪・畝・反・町です。いずれも代表的な近似値として扱います。

坪は面積単位です。長さカテゴリでは扱わず、面積カテゴリの単位として表示します。
