JSON構造をMermaid.jsのフローチャート記法に自動変換する軽量ツール

Convert JSON structures into Mermaid.js flowchart syntax

広告枠

JSONを貼り付けると、Mermaid対応MarkdownやMermaid Live Editorで使える flowchart TD 記法を生成します。処理はブラウザ内で行います。

Paste JSON to generate flowchart TD syntax for Mermaid-compatible Markdown or Mermaid Live Editor. The conversion runs in your browser.

入力上限：300KB / 深さ制限：12階層 / 配列は最大50件まで展開。大きいJSONや深いネストは分割してください。

Limits: 300KB input / 12 levels deep / up to 50 array items. Split large or deeply nested JSON before converting.

よくある質問

FAQ

Q. JSONはサーバーに送信されますか？

Q. Is my JSON sent to a server?

いいえ。変換処理はブラウザ内で行います。ただし広告・解析タグは読み込まれる場合があります。

No. Conversion runs in your browser. Ad and analytics tags may still be loaded.

Q. Mermaidの図をこのページで表示できますか？

Q. Can this page preview the Mermaid diagram?

初期版ではMermaidコード生成が中心です。表示確認はMermaid Live Editorなどを使ってください。

This version focuses on code generation. Use Mermaid Live Editor or a Mermaid-compatible Markdown viewer for previewing.

Q. 大きいJSONに対応していますか？

Q. Does it support large JSON?

300KBまでを目安にしています。大きいJSONや深いネストは分割してください。

The current input limit is 300KB. Split large or deeply nested JSON before converting.

Q. 配列はどう扱われますか？

Q. How are arrays handled?

index付きノードとして展開します。大きい配列は最大50件まで展開し、それ以上は省略します。

Arrays are expanded as index nodes. Large arrays are expanded up to 50 items, then omitted with a notice.

Q. Mermaid Live Editorに貼っても安全ですか？

Q. Is it safe to paste into Mermaid Live Editor?

外部サイトへ貼り付けることになるため、機密情報を含むコードは貼らないでください。

It is an external site, so do not paste code that contains or was generated from confidential information.
