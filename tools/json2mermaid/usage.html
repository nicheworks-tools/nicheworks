JSON2Mermaid Lite｜使い方ガイド

1. このツールでできること（Purpose）

JSONを貼るだけで、その構造を自動解析し、 Mermaid.js の flowchart TD 記法として出力するツールです。

Mermaid Live Editor やドキュメントツールへコピペするだけで、 JSON構造を視覚的に確認できます。

2. 使い方（How to Use）

  • ① JSON入力欄に解析したいJSONを貼り付けます。
  • ② 「変換する」ボタンを押します。
  • ③ 下部にMermaid記法が自動生成されます。
  • ④ 「コピー」でクリップボードに保存できます。
  • ⑤ Mermaid Live Editor に貼り付ければ図を確認できます。

3. 対応 / 非対応（Supported / Unsupported）

対応しているもの

  • オブジェクト（入れ子対応）
  • 配列（indexをノードとして扱う）
  • プリミティブ（文字列 / 数値 / 真偽値）

対応していないもの

  • 循環参照を含むJSON
  • 特殊型（Date / Function / Symbolなど）
  • 極端に巨大なJSON（ブラウザ性能に依存）

4. エラーについて（Error Types）

JSON構文エラーがある場合、「JSONの構文が正しくありません」と表示されます。

大きいJSONの場合、変換に時間がかかることがあります。

5. 出力形式（Output Format）

出力は Mermaid.js の flowchart TD 形式です。

flowchart TD
  node_0["root"]
  node_1["user"]
  node_0 --> node_1
  node_2["name"]
  node_1 --> node_2

6. よくある質問（FAQ）

Q. データはサーバーに送られますか？

A. すべてブラウザ内で処理され、外部送信は一切ありません。

Q. Mermaid記法はどこで使えますか？

A. Mermaid Live Editor（公式）や技術ブログで利用できます。

Q. JSONが大きすぎるとどうなりますか？

A. 変換に時間がかかり、ブラウザが一時停止する場合があります。
