CSV Tidy

列の並べ替え / 削除 / ヘッダ変更 / トリム / 全半 / UTF-8 変換
Tool

最短で使う（30秒）

  1. CSVを読み込む
  2. 不要列を「除外」
  3. 列名を直す / 並べ替える
  4. 必要なら「トリム」「空白正規化」「全半」をON
  5. 出力設定（区切り文字/クォート）を合わせてDL

よくある詰まり

  • 文字化け：Excelで開く前提なら UTF-8(BOM) をONに。
  • 列ずれ：区切り文字が違う可能性。入力の区切りをTAB/セミコロンに切替。
  • 全半変換が効かない：「適用範囲」を「選択した列のみ」にしている場合、列ごとの「変換対象」をONに。

このツールはブラウザ内で処理され、ファイルは外部に送信されません。
