      FAQ
      
      
      このツールは安全を保証しますか？
      Does this tool guarantee safety?
      いいえ。危険パターンを検出する補助ツールです。最終的な実行判断は、内容を確認したうえで行ってください。
      No. It is a helper that detects risky patterns. Review the command before deciding whether to run it.
      
      
      コマンドは送信されますか？
      Are commands uploaded?
      いいえ。入力内容はブラウザ内で処理され、サーバーへ送信・保存されません。
      No. Input is processed in your browser and is not uploaded or stored.
      
      
      PowerShellにも対応していますか？
      Does it support PowerShell?
      一部対応しています。Remove-Item、Invoke-WebRequest、iwr、iex、ExecutionPolicy、Clear-Diskなどを検出します。
      Partially. It detects patterns such as Remove-Item, Invoke-WebRequest, iwr, iex, ExecutionPolicy, and Clear-Disk.
      
      
      Highが出たらどうすべきですか？
      What should I do when High appears?
      そのまま実行せず、対象パス、URL、送信先、権限、代替案を確認してください。迷う場合は実行しないのが安全です。
      Do not run it as-is. Check the path, URL, destination, privilege level, and alternatives. If unsure, do not run it.
