Free checks include

  • PowerShell の Remove-Item / iwr | iex / Clear-Disk などを検出
  • .env、秘密鍵、環境変数表示、curl/scp/rsync/nc による漏洩リスクを警告

  • Warn about .env, private keys, environment dumps, curl/scp/rsync/nc exfiltration risks
