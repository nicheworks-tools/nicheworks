HowTo（運用テンプレ）

ログを取りながら精度を上げるタイプのツールです。数日分のログがあると、作業ごとの燃費差が見えやすくなります。

前提

予測は手入力ログからの簡易推定です。実際の上限、残量表示、リセット時刻を保証しません。

週枠だけ見る

  1. 週%だけ入力して保存します。
  2. 朝と夜など、同じ日にもう1回保存します。
  3. 週の燃費（%/日）と枠切れETAを確認します。
  4. 燃費が荒い場合は、noteにheavy/lightなどを入れて分けます。

5時間枠も見る

  1. 作業開始前に5h%と週%を保存します。
  2. 作業終了後にもう一度保存します。
  3. 5hの燃費（%/時）を見て、作業の重さを比較します。

プロファイル運用

heavy / light など、よく使う絞り込みを保存しておくと、作業タイプ別の燃費を切り替えやすくなります。同名プロファイルは上書きされます。

リセットを手動で固定する

リセット時刻が分かる場合は、Dashboardの手動リセット欄に入力してください。自動推定よりもズレが少なくなります。

データ操作前の注意

大きな操作の前には、必要に応じてExportで控えを作ってください。Importは現在のログ、プロファイル、設定を置き換えます。
