HowTo（運用テンプレ）

これは “使いながら精度を上げる” タイプのツール。 下のテンプレどおりにログを取ると、数日でペース感が掴める。

テンプレ①：最小運用（週枠だけ）

  1. 週%だけ入力 → 保存（mode/status/model/note は空でもOK）
  2. 同じ日にもう1回、週%を入力 → 保存（最低2ログ）
  3. 週の燃費（%/日）と枯渇ETAを見る
  4. 燃費が荒れるなら、noteに「重い作業/軽い作業」を書き分けてフィルタ

おすすめ：朝/夜で1日2回ログ（週枠の燃費が安定しやすい）

テンプレ②：短期運用（5時間枠も見る）

  1. 作業開始前に 5h% / 週% を入力して保存
  2. 作業終了後にもう一度入力して保存
  3. 5hの燃費（%/時）が出る → 「この作業は重い/軽い」が見える

5h%は短期のペース配分に強い（「あと何時間できるか」）

プロファイル運用（おすすめ）

例：2種類だけで良い

  • heavy：大規模PR / 検索多め / 生成量多め
  • light：微修正 / 文章編集 / 小さめPR

ログに note=heavy/light を入れる → フィルタで切替 → Profiles保存。 以降ワンクリックで燃費を切り替えられる。

リセットを手動で固定する（精度UP）

  1. 枠が更新された直後（%が戻った直後）に、その時刻を覚える
  2. Dashboard下部「手動リセット」に次のリセット日時を入力して保存
  3. KPIのサブ行が「手動」になったらOK

リセット日時が確かなら、予測のズレが一気に減る

データ移行（Export/Import）

  1. ExportでJSONを保存（ログ/プロファイル/設定が入る）
  2. 別端末や別ブラウザでImport

localStorageなので “Exportが保険”

困ったら

  • ログが2件未満 → まず2件入れる
  • 燃費が荒い → heavy/lightで分けてフィルタ＋Profiles保存
  • リセットでズレる → 手動リセットを入れる
  • 週枠が主戦場 → 週%だけで運用してOK
