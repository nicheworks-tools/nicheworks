HowTo (operating template)

This tool improves as you log. Follow one of the templates below for a few days to get stable burn profiles.

Template A: minimal (weekly only)

  1. Log Weekly% and Save
  2. Log Weekly% again later the same day (2 logs minimum)
  3. Use burn rate (%/day) + depletion ETA to pace your week
  4. If noisy, tag notes (heavy/light) and filter

Tip: log twice a day (AM/PM)

Template B: short horizon (5h + weekly)

  1. Before work: log 5h% and Weekly%
  2. After work: log again
  3. Use 5h burn (%/h) to compare “heavy” vs “light” sessions

Profiles (recommended)

Keep it simple: just two profiles.

  • heavy: big PR / search-heavy / lots of generation
  • light: small edits / tiny PR

Tag logs with note=heavy/light → filter → save as Profiles for one-click switching.

Pin reset times (accuracy boost)

  1. When you observe a reset, remember the time
  2. Set manual reset datetime on Dashboard and Save
  3. KPI subline shows “manual” when pinned

Export/Import (migration)

  1. Export to JSON (logs + profiles + settings)
  2. Import on another browser/device
