Quick start

  1. On Dashboard, enter the usage percentage (%) you see
  2. Click Save (a few times a day or at work milestones)
  3. After 2+ logs, you’ll see burn rate and ETA

Tips

  • If weekly cap hurts: focus on logging weekly%
  • Burn varies by workload—use notes and model
  • Right after a reset, ETA is noisy—add 2 logs post-reset
  • Move devices: Export JSON → Import
