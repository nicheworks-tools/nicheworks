このツールでできること

Codexなどの表示上の使用率%を手入力で記録し、5時間枠・週枠の燃費と枠切れ時刻を簡易推定します。公式の上限やリセット仕様を判定するものではありません。

重要な注意

実際の使用上限、リセット時刻、残量表示、制限条件は保証しません。サービス側の表示や公式情報を優先してください。

基本の使い方

  1. Dashboardで週%または5h%を入力して保存します。
  2. 最低2回、できれば5回以上ログを取ります。
  3. 燃費と枠切れETAを目安として確認します。
  4. 作業の重さが違う場合は、noteやmodeで分けてフィルタします。

保存されるデータ

使用率ログ、フィルタプロファイル、手動リセット設定、テーマ設定がlocalStorageに保存されます。別端末・別ブラウザには自動で引き継がれません。ブラウザデータを削除すると消えます。

ログ保存と予測処理はブラウザ内で行います。ただし、ページ表示のため広告・解析タグは読み込まれます。

Export / Import

Export JSONにはログ、プロファイル、手動リセット設定が入ります。ImportはこのツールのExport JSONのみを想定し、読み込み後は現在データを上書きします。

Importや全削除の前に、必要なら先にExportしてください。

FAQ

Q. 予測は正確ですか？
いいえ。手入力ログからの簡易推定であり、実際の上限やリセット時刻は保証しません。

Q. 5h%と週%の両方が必要ですか？
片方だけでも保存できます。

Q. データは送信されますか？
ログ保存と予測処理はブラウザ内で行います。広告・解析タグはページ表示のため読み込まれます。
