Contract Cleaner：使い方

契約書・利用規約の「危険っぽい条文」を地雷ワードで素早く拾う（AI不使用／保存なし）。

広告枠（準備中）

このツールでできること

  • 契約書・利用規約などの本文を貼り付けて、リスクになりやすい表現を検出
  • よくある地雷ワード（免責・損害賠償・解除・自動更新・専属管轄など）をハイライト
  • AIを使わない（ルール/辞書ベース）ため、文章を外部に送信しない前提で使える
  • ブラウザ完結・保存なし
ツールを開く English guide

こんなときに使う

  • 契約書を全部読む前に「危ない箇所だけ」先に当たりを付けたい
  • 利用規約の免責・自動更新・解約条件など、よく揉める部分を見落としたくない
  • 相手から来たドラフトの“気になる単語”を一気に洗い出したい

関連キーワード：契約書 危険条文 / 利用規約 地雷 / 免責 損害賠償 / termination auto-renew jurisdiction

使い方

  1. ツール画面を開く
  2. 契約書・利用規約の本文を貼り付ける（必要なら一部だけでもOK）
  3. ハイライトされた箇所（検出語）を起点に、周辺文脈を読む
  4. 気になる条文は、条件・例外・期限・責任範囲を確認する

コツ：検出語がある＝即アウトではなく、まず「条件と例外」を見る。

注意（できないこと・限界）

  • 法律判断や契約の正当性を判定するツールではありません（見落とし防止の補助）。
  • 辞書にない言い回しは検出できない場合があります（言い換え・婉曲表現）。
  • 国/業界/契約種別で重要ポイントが変わるため、必要なら専門家に確認してください。
