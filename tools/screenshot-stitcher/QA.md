# tools/screenshot-stitcher/QA.md
# QA（受け入れ条件）

## A. ヘッダー / レイアウト
- [ ] ヘッダーが固定表示になっていない
- [ ] ヘッダー内にタイトル・短い説明・JP/EN切替がある
- [ ] 本文に重複H1がない
- [ ] ロゴ画像がヘッダーにない
- [ ] ad-top が表示される
- [ ] ad-bottom が表示される
- [ ] カード、余白、フォントの見た目が統一されている

## B. リンク / 寄付導線
- [ ] 寄付導線が2箇所ある
- [ ] 各寄付導線に OFUSE と Ko-fi の2リンクがある
- [ ] OFUSE URL が `https://ofuse.me/nicheworks` に一致
- [ ] Ko-fi URL が `https://ko-fi.com/nicheworks` に一致
- [ ] 問い合わせ 文言がない
- [ ] プライバシー 文言がない
- [ ] 利用規約 文言がない
- [ ] クレジット 文言がない

## C. 入力 / 編集機能
- [ ] クリック選択で複数画像を追加できる
- [ ] D&Dで複数画像を追加できる
- [ ] ペーストで画像追加できる
- [ ] リストにサムネイル・ファイル名・解像度が出る
- [ ] ↑↓で順序変更できる
- [ ] 個別削除できる
- [ ] 全削除できる

## D. 変換 / 出力
- [ ] 幅揃え max/custom/min が機能する
- [ ] 余白トリム ON/OFF が機能する
- [ ] トリム閾値 0-40 が機能する
- [ ] 画像間余白 0-200 が反映される
- [ ] 背景白/透明が反映される
- [ ] プレビュー生成が成功する
- [ ] 高さ20000px超で警告文が表示される
- [ ] PNG出力が成功する
- [ ] WebP出力が成功する
- [ ] JPEG出力が成功する
- [ ] 分割高さ指定でZIP出力が成功する

## E. 言語 / 品質
- [ ] JP/EN切替で主要文言が切替る
- [ ] 320px幅でも操作可能
- [ ] フォーカス表示がある
- [ ] 失敗時にエラー文言が出て再操作できる
- [ ] 「外部送信なし」の文言が表示される
