# tools/screenshot-stitcher/QA.md
# IMG-1 QA（受け入れテスト）

## 1) 共通仕様
- [ ] ヘッダー固定禁止を満たす
- [ ] ロゴ画像が無い
- [ ] ヘッダーに「タイトル＋1行説明」がある（本文H1重複無し）
- [ ] 余計なリンク（Contact/Privacy/Terms/Credits/Donors等）が無い
- [ ] 寄付導線が2箇所（OFUSE/Ko-fi、URL固定）
- [ ] ad-top / ad-bottom がある
- [ ] フッターが最小構成で余計なリンク無し
- [ ] usage/unlock も同トーンで崩れない

## 2) 機能
- [ ] 追加（クリック）
- [ ] 追加（D&D）
- [ ] 並べ替え（↑↓）
- [ ] 削除（個別/全削除）
- [ ] 幅揃え（最大/指定）
- [ ] 簡易トリム（ON/OFF、削りすぎない）
- [ ] Per-image trim（Top/Bottom）が出力に反映
- [ ] Redaction（黒塗り/モザイク）を追加できる
- [ ] Redactionが出力に焼き込まれる
- [ ] プレビュー生成が安定（進捗/Busy）

## 3) 出力/巨大対応
- [ ] PNG出力成功
- [ ] 無料制限：枚数10/高さ12000px（固定値どおり）
- [ ] 制限超過時に落ちない＆unlock導線
- [ ] Proで分割出力ができる（連番PNG）

## 4) UI/UX
- [ ] 320pxでも操作できる（横スクロール地獄にならない）
- [ ] 太字乱用がない（本文通常）
- [ ] 余白が過密/スカスカでない
- [ ] フォーカスが見える（キーボード操作）
- [ ] エラー表示が分かる（画像未対応、メモリ不足など）

## 5) プライバシー
- [ ] 画像の外部送信なし（Networkでupload無し）
- [ ] third-partyは広告/計測スクリプトのみ（仕様どおり）
