よくある質問

FAQ

このLPコピーをそのまま公開できますか？Can I publish this LP copy as-is?

たたき台です。広告表現、価格、実績、法令、媒体ポリシーを必ず確認してください。

Treat it as a draft. Check claims, pricing, proof, regulations, and ad platform policies before publishing.

入力内容は送信されますか？Is my input sent anywhere?

LP骨子生成処理はブラウザ内で行います。ただしページ表示のため広告・解析タグは読み込まれます。

The LP outline is generated in your browser. Ad and analytics tags are still loaded for the page.

数値実績を入れてもよいですか？Can I enter numerical claims?

根拠がある場合のみ入れてください。根拠のない効果・実績表現は避けてください。

Only use numerical claims when they are substantiated. Avoid unsupported performance or proof claims.

HTML出力は安全ですか？Is the HTML export safe?

HTML出力では入力値をエスケープして生成します。ただし公開前に内容とリンク先を確認してください。

The HTML export escapes input values. Still review the content and destination links before publishing.

どの業種でも使えますか？Can this be used for every industry?

医療、金融、投資、美容、健康食品、求人などは特に規制確認が必要です。

Medical, finance, investment, beauty, supplements, and hiring pages require extra regulatory checks.
