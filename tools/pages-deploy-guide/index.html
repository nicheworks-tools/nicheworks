FAQ

このチェックだけでデプロイ成功を保証できますか？ Does this checklist guarantee a successful deployment?

いいえ。公開前確認の補助です。実際のビルドログ、表示、DNS/SSL、公式ドキュメントを確認してください。

No. It is a pre-deploy aid. Always check actual build logs, rendered pages, DNS/SSL status, and official documentation.

入力内容は送信されますか？ Is my input sent anywhere?

チェックリスト生成処理はブラウザ内で行います。ただしページ表示のため広告・解析タグは読み込まれます。

Checklist generation runs in your browser. Advertising and analytics tags may still load for page display.

Cloudflare PagesとGitHub Pagesは同じですか？ Are Cloudflare Pages and GitHub Pages the same?

いいえ。build command、output directory、ドメイン設定、SPA対応などで確認点が異なります。

No. Build command, output directory, domain setup, and SPA handling differ.

カスタムドメインが反映されない場合は？ What should I check when a custom domain does not work?

DNSレコード、SSL状態、CNAME、TTL、プラットフォーム側のドメイン設定を確認してください。

Check DNS records, SSL status, CNAME, TTL, and the platform-side domain setting.

白画面やCSS読み込み失敗は何を見ればよいですか？ What should I check for a blank page or missing CSS?

base path、相対パス、output directory、ビルド後のファイル配置、ブラウザコンソールを確認してください。

Check the base path, relative paths, output directory, built file layout, and browser console.
