Report mode status

このページは Earth Map Suite v0.1 metadata-only MVP の状態を共有用に整理します。実測降水量レポートではありません。

Metadata layer

  • Storm: metadata status connected
  • Compare: Period A/B metadata status connected
  • Card: point-area metadata status connected
  • Sampling status: metadata_only

Important limitation

  • Visible maps, charts, summaries, and CSV numeric values are synthetic preview.
  • Raster precipitation values are not sampled yet.
  • This is not a real precipitation map product.

Storm example

bbox=139.5,35.4,140.0,35.9
start=2025-08-01
end=2025-08-03
preset=low
sampling_status=metadata_only

Compare example

bbox=139.5,35.4,140.0,35.9
A=2025-08-01 to 2025-08-03
B=2025-08-04 to 2025-08-06
A/B/Diff remain synthetic_preview

Card example

lat=35.68
lon=139.76
start=2025-08-01
end=2025-08-03
metadata_bbox≈139.51,35.43,140.01,35.93
point values remain synthetic_preview

Endpoint limits

  • low: up to 14 days
  • mid: up to 7 days
  • Large requests return visible errors.
