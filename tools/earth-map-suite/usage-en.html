1. What this tool does

Summarize map view requirements and generate a shareable memo.

2. Inputs

  • BBox: specify the region as “lon,lat,lon,lat”.
  • Start / end date: the period to review.
  • Preset: choose low / mid / detail.
  • Frames (storm): required in storm mode.
  • Target area: the region or location to review.
  • Layers to verify: roads, terrain, shelters, and more.
  • Notes: additional instructions for reviewers.

3. Preview output

Click “Generate preview” to create a formatted memo. Copy it to share with others.

4. Disclaimer

This page does not verify map accuracy. Always check official sources.

5. Free-ops limits

  • BBox span: up to 5° × 5°.
  • Date span: up to 31 days.
  • Presets: low / mid / detail only. Use low for wider areas.
  • Storm frames: up to 48.

If a limit is exceeded, you will see error_code: limit_exceeded.
