1. What this tool does

Summarize map view requirements and generate storm / compare / card review memos.

2. Quick steps (5)

  1. Select a mode (storm / compare / card).
  2. Enter the area (BBox or lat/lon) and the date range.
  3. Add layers, notes, and review context.
  4. Click “Generate preview” to build the memo.
  5. Copy or download the text, PNGs, or CSVs.

3. Inputs

  • BBox: specify the region as “lon,lat,lon,lat” (storm / compare).
  • Latitude / longitude: point selection (card).
  • Start / end date: the period to review (compare has A/B ranges).
  • Preset: choose low / mid / detail.
  • Frames (storm): required in storm mode.
  • Target area: the region or location to review.
  • Layers to verify: roads, terrain, shelters, and more.
  • Notes: additional instructions for reviewers.

4. Reading the preview

  • storm: a frame slider and cumulative view appear.
  • compare: A/B/diff cards and CSV outputs appear.
  • card: a point preview and a mini time-series chart appear.

All outputs are placeholders for planning, not observed data.

6. Free-ops limits

Item Limit Notes
BBox Up to 5° × 5° Use low for wider areas
Date span Up to 31 days Compare is per A/B range
Preset low / mid / detail Other values error out
Storm frames Up to 48 Above 48 will be thinned

If a limit is exceeded, you will see error_code: limit_exceeded.

7. Troubleshooting

Q. The preview is blank.
A. Make sure required fields (mode, dates, BBox/lat-lon) are filled.
Q. Compare cards do not show.
A. Switch to compare mode and fill both A/B date ranges.
Q. I see an error code.
A. Check the limits table for the free-ops caps.
