Export mode status

このページは、現在の metadata-only MVP を前提に、共有・記録用のCSV/Markdownテンプレートを生成します。実測降水量のexportではありません。

Export guardrails

  • CSV numeric values must stay synthetic_preview.
  • Raster precipitation values are not sampled.
  • Metadata status may be reachable, but maps/charts are not observed precipitation.

Included metadata fields

  • mode
  • data_type=synthetic_preview
  • metadata_status
  • dataset_id
  • sampling_status=metadata_only
  • generated_at
