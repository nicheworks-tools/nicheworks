使い方｜旧字体 ⇄ 新字体 変換ツール

このページでは 旧字体 ⇄ 新字体 変換ツール の詳細な使い方・注意点を解説します。

1. このツールは何をする？

旧字体（舊・學・體など）を新字体に一括変換したり、 新字体（旧・学・体など）を旧字体へ戻すツールです。 完全ローカルで動作します。

2. 変換方向について

  • 旧 → 新 … 古い文字を現代の文字へ変換
  • 新 → 旧 … 現代の文字を旧字体へ変換（複数候補は先頭を採用）

3. 操作方法

  1. 変換方向（旧→新 / 新→旧）を選ぶ
  2. テキストを入力欄に貼る
  3. 「変換する」を押す
  4. 結果欄をコピーできる

4. 収録字数

当ツールの辞書は以下の数を収録しています：

  • 旧字体 → 新字体：836字
  • 新字体 → 旧字体：457字
  • 総ユニーク字数：1,293字

5. プライバシー

入力文字は一切サーバーに送信されません。 すべてブラウザ内（ローカル）で処理されます。

6. 注意点

  • 異体字（康煕字典体など）は変換対象外のものがあります
  • 複数の旧字体が存在する場合、新→旧は「最も一般的な形」を採用
  • 人名固有の特殊字体（高﨑・吉﨑など）は辞書収録の範囲内で変換
