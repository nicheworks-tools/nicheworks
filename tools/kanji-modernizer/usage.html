1. このツールでできること

旧字体（舊・學・體など）を新字体へ、または新字体を旧字体候補へ変換できます。人名・地名・古い文書の確認を補助する辞書ベースのツールです。

2. 使い方

  1. 変換方向を選びます。
  2. テキストを貼り付けます。
  3. 必要ならURLやコードの除外をONにします。
  4. 「変換する」を押し、結果と置換一覧を確認します。

3. 対応 / 非対応

登録済みの文字だけを1文字単位で置換します。未登録の異体字、文脈依存の使い分け、正式表記の保証には対応しません。重要な用途では必ず原本や公式情報をご確認ください。

  • 旧→新：登録済み旧字体を新字体へ変換します。
  • 新→旧：複数候補がある場合、同一文字ではない候補を優先します。
  • URL、コード、ASCII文字列はオプションで除外できます。

4. プライバシーと外部タグ

変換処理はブラウザ内で行われ、入力本文はNicheWorksのサーバーへ送信しません。ただし、このページでは運営のためにAdSense、GA4、Cloudflare Web Analyticsなどの広告・アクセス解析関連の仕組みが使われる場合があります。

5. FAQ

すべての旧字体を変換できますか？

いいえ。辞書に未登録の文字は変換されません。

正式な表記確認に使えますか？

補助用途です。公式書類や原本の確認を優先してください。

入力本文は保存されますか？

ツール側で入力本文を保存する機能はありません。
