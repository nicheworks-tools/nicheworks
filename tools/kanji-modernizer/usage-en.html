1. What this tool does

Kanji Modernizer converts old-style kanji such as 舊, 學, and 體 into modern forms, or converts modern kanji into old-form candidates. It is a dictionary-based helper for reviewing names, placenames, shrine or temple names, and historical text.

2. How to use

  1. Select the conversion direction.
  2. Paste the text you want to check.
  3. Optionally exclude URLs, code blocks, and ASCII strings.
  4. Press “Convert” and review the result and replacement list.

3. Supported / unsupported

The tool only replaces registered characters one by one. It cannot guarantee legal names, official records, context-specific forms, or unregistered variants. For important use cases, always check the original or official source.

  • Old → Modern: converts registered old forms into modern forms.
  • Modern → Old: when multiple candidates exist, it prefers a candidate that differs from the input character.
  • URLs, code blocks, and ASCII strings can be excluded.

4. Privacy and external tags

The conversion runs in your browser, and the input text is not uploaded to a NicheWorks server. This page may still load advertising and analytics tools such as AdSense, GA4, and Cloudflare Web Analytics for site operation.

5. FAQ

Can it convert every old kanji?

No. Characters that are not in the dictionary are left unchanged.

Can I use the result as an official spelling?

No. Use this as a helper and verify official documents or original sources.

Is the input text saved?

The tool does not provide a feature that stores your input text.

6. Disclaimer

This tool is a support utility for old/modern kanji conversion. It does not guarantee accuracy or suitability for official procedures.
