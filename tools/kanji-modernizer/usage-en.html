Usage Guide｜Kanji Modernizer Lite

This page explains how to use Kanji Modernizer Lite and the rules behind its conversion.

1. What does this tool do?

Converts Old Kanji (舊・學・體 etc.) to Modern Kanji, and converts Modern Kanji back into their Old forms. All processing is done locally in your browser.

2. Conversion directions

  • Old → Modern: Convert traditional forms into simplified forms.
  • Modern → Old: Convert simplified forms into old/traditional forms. If multiple candidates exist, the first one is used.

3. How to use

  1. Select the conversion direction
  2. Paste your text in the input area
  3. Press “Convert”
  4. Copy the output text

4. Dictionary size

The tool contains the following:

  • Old → Modern: 836 characters
  • Modern → Old: 457 characters
  • Total unique characters: 1,293

5. Privacy

No data is uploaded. 100% client-side processing.

6. Notes

  • Some rare variants (Kangxi forms) are not included
  • Modern → Old picks the most common old form
  • Special family-name variants are supported only if in the dictionary
