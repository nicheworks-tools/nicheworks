旧字体・新字体 一括変換ツール Kanji Modernizer (Old ⇄ Modern)

旧→新／新→旧を辞書ベースで一括変換します。入力された文字はブラウザ内のみで処理されます。 Convert between old and modern kanji locally in your browser using a curated dictionary.

このツールがすること・しないこと What this tool does / does not do

  • 辞書に登録された旧字・新字のみを1文字単位で置換します。
  • 文脈や固有名詞の正確性は保証しません。最終確認はご自身で行ってください。
  • Replaces characters based on the dictionary (single-character mapping only).
  • It does not validate context or proper nouns. Please review the output.

収録字数：旧→新 字 / 新→旧 字（ユニーク 字）

Dictionary size: Old→New , New→Old (Unique )
