Kanji Modernizer：使い方

旧字体⇄新字体を辞書ベースで一括変換します（ローカル処理）。

広告枠（準備中）

このツールでできること

  • 旧字体（旧字）→ 新字体（現代の漢字）を一括変換
  • 逆方向：新字体 → 旧字体 にも対応（対応している場合）
  • 姓名・地名・古文書・寺院名などの「旧字混じり」を素早く整える
  • 完全ローカル処理・保存なし
ツールを開く English guide

こんなときに使う

  • 役所・登記・相続などで、旧字を「通る表記」に寄せたい
  • 古い名簿・文書の文字を現代漢字に寄せて検索しやすくしたい
  • 地名・寺社名・人名の旧字体を、普段の入力に合わせたい

関連キーワード：旧字体 変換 / 新字体 変換 / 旧字 一括 / kanji old to new / kyūjitai shinjitai converter

使い方

  1. 変換したい文章を貼り付ける
  2. 変換方向（旧→新 / 新→旧）を選ぶ
  3. 出力をコピーして使う

コツ：固有名詞は表記揺れがあるので、結果は元の資料と照合して使ってください。

注意（できないこと・限界）

  • 辞書に無い旧字や異体字は変換されない場合があります。
  • 同じ文字でも文脈で最適が変わる場合があります（人名・地名など）。
  • 公的手続きでは「正確な表記」が必要な場合があります。最終的に原本表記を優先してください。
