よくある質問

FAQ

画像はサーバーに送信されますか？

Are images uploaded to a server?

いいえ。画像の読み込み、再圧縮、サイズ確認はブラウザ内で処理されます。

No. Loading, re-compression, and size checks are handled in your browser.

PNGを圧縮できますか？

Can this compress PNG files?

PNGを読み込んでJPEGまたはWebPに変換し、生成後サイズを比較できます。PNGの可逆圧縮出力は行いません。

You can load a PNG and compare JPEG/WebP output sizes. Lossless PNG optimization is not supported.

JPEGにすると透明部分はどうなりますか？

What happens to transparent areas when using JPEG?

JPEGは透過に対応しないため、このツールでは透明部分を白背景として保存します。

JPEG does not support transparency, so this tool saves transparent areas on a white background.

生成後サイズが元画像より大きくなることはありますか？

Can the generated file become larger than the original?

あります。元画像がすでに十分圧縮されている場合、再変換で大きくなることがあります。

Yes. If the original is already well compressed, re-compression can create a larger file.

HEICやRAWに対応していますか？

Are HEIC and RAW supported?

対応していません。PNG、JPEG、WebPなど、ブラウザで読み込める静止画像を対象にしています。

No. This tool targets static images your browser can read, such as PNG, JPEG, and WebP.
