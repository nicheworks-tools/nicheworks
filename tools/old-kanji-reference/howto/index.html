旧字 → 現代表記を、検索して即コピペできるリファレンス。

このツールでできること

  • 一覧旧字体と新字体（現代表記）をペアで表示
  • 検索入力と同時に即フィルター（候補を絞る）
  • コピー必要な文字/表記をすぐコピー
  • 用途戸籍・古文書・寺社名・人名などの表記確認
ツールを開く English guide

よくある症状

  • 古い書類の漢字が読めない/入力できない
  • 戸籍・登記・寺社名で「旧字のまま」必要と言われた
  • 検索すると候補が多すぎて、どれが正しいか分からない
  • 旧字をコピペしたいだけなのに、変換候補が出ない

関連キーワード：旧字体 一覧 / 旧字 変換 / 旧字体 検索 / 旧漢字 コピペ / 戸籍 旧字

使い方（最短3ステップ）

  1. 検索欄に「現代の字」または「読み/キーワード」を入力する
  2. 候補から該当のペア（旧字 ↔ 新字）を見つける
  3. コピー機能で必要な文字をコピーして貼り付ける

コツ：まずは「新字（普段の字）」で探す→見つかった旧字をコピペ、が最短。

注意（できないこと・限界）

  • フォント環境により、旧字が表示できない端末があります。
  • このツールは「変換の確定」ではなく、表記候補の参照です。
  • 公的書類は提出先の指定が優先されます（最終表記は提出先の指示に従ってください）。
