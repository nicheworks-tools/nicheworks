What it does

  • Extracts title / description / Open Graph (og:title, og:description, og:image)
  • Checks canonical and other key tags (depending on output)
  • Useful before sharing on social platforms or during site migrations
  • Browser-only, no storage
Open the tool 日本語ガイド

How to use

  1. Paste the URL
  2. Run extract (or auto-fetch)
  3. Compare title/description/OGP/canonical with what you expect

Tip: social platforms cache previews. Even after you fix metadata, the preview may update later.

Notes (limits)

  • Some URLs can’t be fetched due to restrictions (login, blocks).
  • Dynamic sites may return different results depending on environment.
  • This tool can’t force-refresh social preview caches.
