MetadataSnap：使い方

URLを貼って「今そのページが何を名乗っているか（メタ情報）」を一瞬で確認。

広告枠（準備中）

このツールでできること

  • 抽出title / description / OGP（og:title, og:description, og:image）
  • 確認canonical / twitter card など（ツールの出力項目に依存）
  • 用途SNS共有前のOGP確認、LP/ブログのメタ設定チェック、サイト移行時の差分確認
  • ブラウザ完結・保存なし
ツールを開く English guide

よくある症状

  • X/LINE/Discordでリンクを貼ったら、タイトルや画像が変（古い/違う）
  • OGP画像が出ない / サムネが本文の画像になってしまう
  • canonicalが意図しないURLになっていて、SEOが崩れそう
  • 多言語ページでog:titleが混在している気がする

関連キーワード：OGP 確認 / meta title description check / og:image 取得 / canonical 確認 / SNS サムネ 変

使い方（最短3ステップ）

  1. チェックしたいURLを貼り付ける
  2. 抽出ボタンを押す（または自動取得）
  3. title / description / OGP / canonical を見て、意図通りか確認する

コツ：SNSはキャッシュが残る。Metaは直しても「反映が遅い」ことがあるので、まずは現在値を把握する。

注意（できないこと・限界）

  • アクセス制限（ログイン必須/ブロック）されているURLは取得できない場合があります。
  • サイト側が動的に出し分ける場合、環境によって結果が変わることがあります。
  • SNS側のキャッシュ更新はこのツールでは操作できません。
