URL Title Collector

複数URLのページタイトルをまとめて取得し、CSV/TSVでコピーできます

広告枠（準備中）

使い方はこちら

URLを1行ずつ貼り付けると、各ページの <title> を取得して一覧化します。結果はCSV/TSV形式でコピーできます。

重要な注意

このツールはページタイトル取得のため、入力したURLをNicheWorksのWorkerへ送信します。機密URL、社内URL、ログイン後ページ、個人情報を含むURLは入力しないでください。

短縮URLや不審なURLは、先に Redirect Unwrapper で確認することを推奨します。

http / https のURLのみ処理します。重複URLや不正URLはWorkerへ送信せず、結果に理由を表示します。

FAQ

入力したURLは送信されますか？

はい。ページタイトル取得のため、入力URLはNicheWorksのWorkerへ送信されます。

社内URLやログイン後ページを入れてもよいですか？

推奨しません。機密URL、社内URL、ログイン後ページ、個人情報を含むURLは入力しないでください。

CORSエラーは起きますか？

ブラウザから直接ではなくWorker経由で取得しますが、相手サイトの制限・遮断・タイムアウトにより失敗する場合があります。

短縮URLの最終URLも分かりますか？

このツールはタイトル取得が目的です。追跡リンクや短縮URLの確認はRedirect Unwrapperを使ってください。
