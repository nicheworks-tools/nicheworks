URL Title Collector の使い方

URL Title Collector は、複数URLを1行ずつ貼り付け、各ページの <title> タグ を取得して一覧化する軽量ツールです。

重要：タイトル取得にはNicheWorksの外部Workerを使用します。入力したURLはWorkerへ送信されます。機密URL、社内URL、ログイン後ページ、個人情報を含むURLは入力しないでください。

1. 使い方（3ステップ）

  • テキストエリアに URL を 1 行ずつ貼り付けます
  • 「タイトルを取得する」を押します
  • URL・タイトル・ステータスが一覧で表示されます

2. 入力ルール

  • 処理対象は http: / https: のURLのみです
  • 最大50件まで処理します
  • 重複URL、不正URL、javascript: / data: / file: などはWorkerへ送信しません
  • 短縮URLや不審なURLは、先に Redirect Unwrapper で確認してください

3. ステータスの種類

  • success：タイトル取得に成功
  • no-title：ページ取得は成功したが <title> が見つからない
  • http-error：404 / 500 などのHTTPエラー
  • network-error：通信・DNS・Worker側取得などの失敗
  • timeout：10秒以内に取得できなかった
  • invalid-url / invalid-scheme / duplicate / skipped-limit：Workerへ送信せずスキップ

4. 対応 / 非対応

通常のHTMLページやブログ、ニュース記事などに対応します。PDF、JSON、XML、ログイン必須ページ、JavaScriptで後からタイトルを書き換えるページは取得できない場合があります。

5. CSV / TSV 出力

結果は url / title / status の3列で、CSVまたはTSVとしてコピーできます。CSVはExcel等で扱いやすいようCRLF改行で出力します。

6. FAQ

Q. 入力したURLは送信されますか？
はい。ページタイトル取得のため、入力URLはNicheWorksのWorkerへ送信されます。

Q. ページ本文も保存されますか？
このページではタイトル抽出結果だけを表示します。ただしWorker側でURL取得が行われます。

Q. 取得できないURLがあるのはなぜですか？
相手サイトの制限、非HTMLレスポンス、通信失敗、タイムアウトなどが原因です。

7. 免責

本ツールは取得結果の正確性を保証しません。機密URLや非公開URLの入力は避け、必要に応じて公式情報をご確認ください。
