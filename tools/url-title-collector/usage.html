URL Title Collector の使い方

URL Title Collector は、複数のURLをまとめて貼り付けるだけで、 <title> タグを高速取得できる超軽量ツールです。

1. 使い方（3ステップ）

  • ① テキストエリアに URL を 1 行ずつ貼り付けます
  • ② 「タイトルを取得する」を押します
  • ③ URL とタイトル・ステータスが一覧で表示されます

結果は CSV / TSV 形式でコピーできます。リセットボタンで初期状態に戻ります。

2. ステータスの種類

  • success：タイトル取得に成功
  • no-title：ページ取得は成功したが <title> が存在しない
  • http-error：HTTPエラー（404 / 500 など）
  • network-error：CORS / DNS などネットワーク側の問題

3. 対応しているケース

  • 通常のHTMLページ
  • ニュースサイト・ブログなどの静的なページ
  • <title> タグを持つページ

4. 非対応のケース

  • ログインが必要なページ
  • PDF / JSON / XML など HTMLでないURL
  • CSP（Content Security Policy）が非常に強いページ
  • JavaScriptで動的に <title> を変更するサイト

5. CSV / TSV 出力

結果テーブルはそのまま CSV（カンマ） / TSV（タブ） 形式でコピーできます。 Excel / Google Sheets / Notion などに貼り付け可能です。

6. よくある質問（FAQ）

Q：404でもタイトルが返ることがある？
独自の404ページ（Not Found HTML）が返され、そのHTML内の <title> を取得しているためです。

Q：海外サイトで失敗しやすい理由は？
CORS / CSP が強く、Worker 経由でも本文を取得できないためです。

Q：大量URLの上限は？
ブラウザ動作のため、100〜300件程度が安全です（端末性能に依存）。

7. 免責

本ツールはブラウザ内と Cloudflare Worker 上で動作します。 入力URL・結果は保存されません。 取得できないURLが存在する場合があります。
