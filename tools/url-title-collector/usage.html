URL Title Collector の使い方

URL Title Collector は、複数のURLを一括貼り付けするだけで <title> タグの内容を高速に取得できる軽量ツールです。

1. 使い方（3ステップ）

  • ① テキストエリアに URL を 1 行ずつ貼り付けます。
  • ② 「タイトルを取得する」ボタンを押します。
  • ③ 一覧表に URL とタイトル・ステータスが表示されます。

CSV / TSV 形式でコピーでき、リセットボタンで初期化できます。

2. ステータスの意味

  • success：タイトル取得に成功
  • no-title：ページは取得できたが <title> が存在しない
  • http-error：HTTPエラー（404, 500 など）
  • network-error：ネットワーク側の問題（CORS・DNS・接続不可など）

3. 対応しているケース

  • 通常のWebページ（HTML）
  • <title> タグを持つ静的ページ
  • ほとんどのニュース・ブログサイト

4. 非対応のケース（仕様上の限界）

  • ログインしないと閲覧できないページ
  • JSON / XML / PDF など HTML 以外の URL
  • 高度なCSP（Content Security Policy）で保護されたサイト
  • JavaScript動的生成で <title> を後から変更するサイト

5. CSV / TSV 出力について

結果テーブルの内容をそのまま CSV（カンマ区切り）/ TSV（タブ区切り） でコピーできます。 Excel / Google Sheets / Notion などに貼り付けて整理できます。

6. よくある質問（FAQ）

Q：404でもタイトルが返ってくることがある？
サーバーが独自の404ページ（Not Found HTML）を返しており、そのHTML内の<title>を取得しているためです。

Q：海外サイトで失敗する理由は？
CORSやCSPの制限により、Worker経由でも本文を取得できない場合があります。

Q：何件まで一括処理できる？
ブラウザ動作のため、100〜300件程度が安全です（端末性能による）。

7. 免責

本ツールはすべてブラウザ内＋Cloudflare Worker で動作し、 URLリスト・結果は保存されません。 取得できないURLが存在する場合があります。
