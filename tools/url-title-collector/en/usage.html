How to Use URL Title Collector

URL Title Collector is a lightweight tool for pasting multiple URLs and fetching each page's <title> tag into a simple result table.

Important: This tool uses an external NicheWorks Worker to fetch titles. Entered URLs are sent to that Worker. Do not enter private URLs, internal URLs, login-only pages, or URLs containing personal information.

1. Basic Steps

  • Paste URLs into the text area, one URL per line
  • Click Fetch Titles
  • Review the URL, title, and status table

2. Input Rules

  • Only http: and https: URLs are processed
  • The processing limit is 50 entries
  • Duplicate URLs, invalid URLs, and schemes such as javascript:, data:, or file: are not sent to the Worker
  • Check short links or suspicious URLs with Redirect Unwrapper first

3. Status Types

  • success — title fetched successfully
  • no-title — page fetched but no <title> was found
  • http-error — 404 / 500 / other HTTP error
  • network-error — network, DNS, or Worker fetch failure
  • timeout — the request did not finish within 10 seconds
  • invalid-url / invalid-scheme / duplicate / skipped-limit — skipped before sending to the Worker

4. Supported / Unsupported

Standard HTML pages, blogs, and news pages are supported. PDF, JSON, XML, login-required pages, and pages that update titles only after JavaScript execution may fail or return no title.

5. CSV / TSV Output

Results are copied as three columns: url / title / status. CSV output uses CRLF line endings for better spreadsheet compatibility.

6. FAQ

Q. Are entered URLs sent anywhere?
Yes. Entered URLs are sent to a NicheWorks Worker so it can fetch the page title.

Q. Is page content stored?
This page only displays extracted title results. However, the Worker still fetches the URL to retrieve the page title.

Q. Why do some URLs fail?
The target site may block access, return non-HTML content, time out, or fail during network retrieval.

7. Disclaimer

This tool does not guarantee complete or accurate results. Avoid entering private or sensitive URLs and verify important information with official sources.
