セルフ判定 → 次にやること → 中長期計画ドラフト → Markdownプレビューまでを説明します。

1. このツールでできること

荷待ち、納品時間指定、委託、可視化不足、混雑などの状況をYes / No / Unknownで棚卸しし、実務整理用の目安と計画ドラフトを作ります。

法令対応、行政提出、契約義務、荷主責任の有無を判定するツールではありません。

2. 使い方の手順

  1. 各項目を Yes / No / Unknown で選びます。分からない項目は Unknown のままで構いません。
  2. 必要なら「現状メモ」に、困っている状況や社内共有したい前提を書きます。
  3. 「判定・ドラフト作成」を押します。
  4. 判定結果、検出シグナル、次にやること、判定基準、中長期計画ドラフトを確認します。
  5. Markdownプレビューを確認します。Markdown保存とPDF出力はPro機能として扱います。

3. 対応 / 非対応

対応

  • 荷待ち・待機の有無の整理
  • 納品時間帯指定の厳しさの整理
  • 委託有無、可視化不足、混雑の棚卸し
  • 次にやることのチェックリスト生成
  • 中長期計画ドラフトの生成
  • Markdownプレビュー
  • Markdown保存（Pro）

非対応

  • 法令適合の断定
  • 行政提出用書類の作成
  • 契約義務や荷主責任の有無の判定
  • 個別企業に対する法的・制度的な助言
  • PDF出力（Pro予定・未実装）

4. 判定の見方

  • 高：待機・時間指定・委託・混雑・可視化不足などのシグナルが3つ以上
  • 中：シグナルが1〜2個
  • 低：顕著なシグナルが少ない
  • 不明：未入力が多く、判断材料が不足

判定は実務整理の目安です。制度対応や法的判断は、公式情報、専門家、社内法務・物流部門に確認してください。

5. 出力形式

  • 画面表示：判定、根拠、次にやること、判定基準、現状メモ、中長期計画ドラフト
  • Markdownプレビュー：社内メモ、会議準備、課題整理に転記しやすい形式
  • Markdown保存：Pro機能として扱います

現状メモは判定ロジックには使わず、出力メモとして記録します。

6. 入力内容の扱い

入力内容の判定処理はブラウザ内で行います。ただし、ページ表示のため広告・解析タグは読み込まれます。機密情報や個人情報をそのまま貼り付けないでください。

7. FAQ

無料で使える範囲はどこまでですか？

初期版では判定とMarkdownプレビューまで無料で使えます。Markdown保存とPDF出力はPro機能として扱います。

PDF出力はありますか？

初期版では未実装です。Pro予定の機能として扱います。

Unknownが多い場合はどうすればよいですか？

まず現場記録やヒアリングで判断材料を増やす使い方を想定しています。

8. 免責

このツールは物流効率化・荷待ち・納品条件などの状況整理を支援する簡易ツールです。法令対応、行政提出、契約義務、荷主責任の有無を判定するものではありません。

制度対応や法的判断は、国交省・経産省などの公式情報、専門家、社内法務・物流部門に確認してください。
