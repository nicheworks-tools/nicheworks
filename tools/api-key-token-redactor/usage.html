使い方｜API Key & Token Redactor

最終安全チェック用の簡易ガイド

1. できること

  • コード／ログ／.env などから、よくあるAPIキー・トークン風文字列を検出します。
  • Free版では「プレビュー用の伏字（例：sk-12…9abc）」として表示します。

2. 使い方

  1. テキストを貼り付けます（コード・ログ・設定ファイルなど）。
  2. 「検出する / Scan」を押します。
  3. 結果を確認し、必要ならコピーボタンで伏字済みテキストをコピーします。

3. 対応 / 非対応

  • 対応：OpenAI / Stripe / AWS / GitHub / JWT など（ベストエフォート）
  • 非対応：全ての秘密情報の完全検出（保証できません）

4. 免責

本ツールは完全な検出を保証しません。共有前の最終確認は必ず目視で行ってください。
