APIキー・トークン伏字ツールの使い方

30秒ガイド、対応パターン、注意点をまとめています。

30秒ガイド

  1. ログや設定ファイルをコピーしてツールに貼り付けます。
  2. 「伏字にする」を押して出力を確認します。
  3. 出力だけを共有（GitHub/Discord/フォームなど）します。

共有前に検索（Ctrl+F）で sk- や AKIA などを確認すると安心です。

対応パターン

  • Bearer トークン（Authorization ヘッダー / Bearer プレフィックス）
  • JWT（eyJ... 形式）
  • PEM形式の秘密鍵（BEGIN/END PRIVATE KEY）
  • APIキー例: sk-, sk_live_, ghp_, AKIA
  • ラベル付きシークレット（api_key=, token:, password= など）

できないこと

  • 伏字漏れゼロの保証はできません。
  • 画像やスクリーンショット内の情報は検出できません。
  • トークンの有効性チェックや無効化は行いません。

プライバシー

入力内容はブラウザ内で処理され、外部に送信されません。
