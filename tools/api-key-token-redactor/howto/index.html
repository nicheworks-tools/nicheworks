ツールを開く English guide

※最終確認は必ず人間が目視してください（伏字漏れゼロは保証できません）。

よくある症状（検索される言い方）

  • GitHub Issue / Discord にログを貼りたいが、APIキーが混ざってないか不安
  • Authorization: Bearer ... や api_key=... を消し忘れそう
  • .env / JSON / curlコマンドの一部を安全に共有したい

関連キーワード：api key redaction, token masking, bearer token, authorization header, secret sanitize, log sharing

このツールがやること（安全な言い方）

  • 「APIキー/トークンに見えやすいパターン」を検出し、伏字化を手伝う
  • 長いログでも「怪しい箇所」を見つけやすくする
  • 処理はブラウザ内（保存なし）

例：sk-**** / Bearer **** / api_key=**** のように置き換えて共有しやすくします。

使い方（最短3ステップ）

  1. ツール画面を開く
  2. 共有したいテキストを貼る
  3. 伏字後のテキストをコピーして貼る

重要：伏字後も「鍵っぽい文字列」が残っていないか、必ず目視で確認してください。
