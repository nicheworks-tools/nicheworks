What it does

How to use

  1. Paste the SQL you plan to run (multiple statements OK)
  2. Select DB/options if available
  3. Run the check and review risk + highlights

Notes

  • This tool does not execute SQL and does not see your database.
  • “Safe” does not guarantee correctness—review execution plan/transaction.
