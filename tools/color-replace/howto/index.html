Color Replace Lite：色置換ガイド

指定色を許容範囲で置換し、PNGで保存する方法を紹介します。

広告枠（準備中）

できること

  • 画像内の特定色を別の色に置換
  • 許容範囲（近い色の扱い）をスライダーで調整
  • 結果をPNGで保存（透明背景は保持）
  • すべてブラウザ内処理で完結（ローカルのみ）
ツールを開く English guide

使い方

  1. 画像をアップロードする
  2. 「置換対象の色」をカラーピッカーで選ぶ（または画像をクリック）
  3. 「置換後の色」を選ぶ
  4. 許容範囲を調整し、置換を適用 をクリック
  5. プレビューを確認して PNGで保存

処理はブラウザ内のみ。画像はサーバーに送信されません。

ベストプラクティス

  • 写真のように色が連続する画像は、許容範囲を小さくするとにじみを抑えられます。
  • ロゴ・アイコンのようなベタ塗りは、許容範囲を少し広げると置換が安定します。
  • 大きな画像は自動で縮小してプレビューします（ダウンロードも同サイズ）。

透明背景について

PNGのアルファチャンネルは保持されるため、背景透過のまま色置換できます。
