Color Replace Lite：色置換ガイド

画像内の「この色だけ」を別の色に置換（ズーム/許容範囲/PNG保存）。

広告枠（準備中）

このツールでできること

  • 画像からスポイトで色を抽出し、指定色に置換
  • 置換色は HEX / RGB / プリセットから指定
  • 許容範囲（近い色の扱い）を調整して、置換の効き具合をコントロール
  • ズームしながら細かい部分を選び、結果をPNGで保存
ツールを開く English guide

こんなときに使う

  • ロゴやアイコンの色だけを差し替えたい（背景は触りたくない）
  • スクショの一部（マーカー色など）を別色に変えたい
  • 複数の色バリエーションを試して比較したい

関連キーワード：色 置換 / 指定色 変換 / スポイト / HEX RGB / PNG 保存 / color replace / replace color in image

使い方

  1. 画像を読み込む
  2. スポイトで「置換したい元の色」を選ぶ（From）
  3. 置換先の色（To）を HEX/RGB/プリセットで指定
  4. 必要ならズーム＆許容範囲を調整して、置換結果を確認
  5. 問題なければPNGで保存

処理はブラウザ内のみ。画像はサーバーに保存しません。

注意（できないこと・限界）

  • 写真のように色が連続的に変化する画像は、置換が「にじむ」ことがあります（許容範囲で調整）。
  • 完全な境界の自動認識（例：背景だけを賢く除外）は行いません。
  • 元画像がJPEGだと、圧縮ノイズで“同じ色に見えても数値が違う”ことがあり、置換が部分的になる場合があります。
