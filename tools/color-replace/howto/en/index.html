Color Replace Lite: Guide

Replace “this exact color” with another one (eyedropper, tolerance, zoom, PNG export).

What it does

  • Pick a source color with an eyedropper and replace it with a target color
  • Set the target color via HEX, RGB, or presets
  • Adjust tolerance to control how much “similar colors” are replaced
  • Zoom for precise picking, then export as PNG
Open the tool 日本語ガイド

Use this when

  • You want to swap a logo/icon color without touching the background
  • You need to change a highlight/marker color in a screenshot
  • You want to quickly try multiple color variations

Keywords: replace color in image, color replacement, eyedropper, hex rgb, png export

How to use

  1. Load an image
  2. Pick the “From” color with the eyedropper
  3. Set the “To” color (HEX/RGB/preset)
  4. Adjust zoom and tolerance until the result looks right
  5. Export as PNG

Runs locally in your browser. No server-side image storage.

Notes (limits)

  • Photos with smooth gradients may look “smudged” after replacement (tune tolerance).
  • This tool does not do smart segmentation (e.g., “only background”).
  • JPEG compression noise can make “the same-looking color” differ per pixel, causing partial replacement.
