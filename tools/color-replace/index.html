Color Utility

Color Replace Lite

画像内の指定色をクリックで取得し、許容範囲を調整して別の色に置換できるブラウザツールです。

広告枠（上）

PNG/JPEG/WebP画像を読み込み、画像上で選んだ色を別の色に置換してPNGで保存できます。

  1. 画像を選ぶ
  2. 変えたい色を画像上でクリック
  3. 置換後の色と許容範囲を調整
  4. 置換を適用
  5. PNGで保存

画像の読み込み・色置換処理はブラウザ内で行います。画像ファイルはアップロードしません。ただしページ表示のため広告・解析タグは読み込まれます。

入力：PNG / JPEG / WebP。出力：PNGのみ。Canvas経由のため、EXIFなどのメタデータは保持されない場合があります。

よくある質問

画像はアップロードされますか？

いいえ。画像の読み込みと色置換処理はブラウザ内で行います。ただしページ表示のため広告・解析タグは読み込まれます。

元画像は変更されますか？

いいえ。元画像ファイルは変更せず、置換後のPNGを新しく保存します。

大きい画像はそのまま保存されますか？

いいえ。大きい画像は処理負荷を下げるため最大4MP程度に縮小され、保存画像も縮小後サイズになります。

JPEGやWebPで保存できますか？

初期版ではPNG保存のみです。

透明部分は維持されますか？

透明度は基本的に維持しますが、見え方は背景や画像ビューアによって変わる場合があります。
