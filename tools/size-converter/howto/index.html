ツールを開く English guide

よくある症状

  • JPの「M」や「27cm」が、US/EUだと何番か分からない
  • ショップのサイズ表がバラバラで混乱する
  • 靴だけ合わず返品コストが高い
  • 男女・国で表記が違って選べない

関連キーワード：JP US サイズ換算 / EU サイズ対応表 / 海外通販 サイズ 迷う / 靴 27cm US / clothing size chart

使い方（最短3ステップ）

  1. カテゴリ（服/靴）と性別を選ぶ
  2. 自分のJPサイズ（例：M、27cm など）を選ぶ
  3. 表示されたUS/EUサイズを、そのまま購入画面に入力/選択する

コツ：同じ「US 9」でもブランド差があるので、迷うときは「実寸（cm）」とレビューも併用。

注意（できないこと・限界）

  • ブランド固有の「小さめ/大きめ」までは補正できません。
  • 幅（ワイズ）や足型は別問題なので、靴はレビュー参照推奨です。
  • 各国サイズ表は目安で、最終判断はショップの公式表が優先です。
