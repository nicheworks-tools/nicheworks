このツールでできること

  • 化粧品の成分表示（全成分）を貼り付けて、主要成分の説明を表示
  • 気になる成分の「ざっくり役割」を短文で把握できる
  • ブラウザ完結・保存なし
ツールを開く English guide

こんなときに使う

  • 買う前に「どんな成分が入ってるか」をさっと見たい
  • いつも肌に合わない成分が入っていないか、名前を確認したい
  • 英語表記の成分を見て、用途の当たりを付けたい

関連キーワード：化粧品 成分 チェック / ingredient checker / INCI / 敏感肌 成分 / 成分表示 読み方

使い方

  1. パッケージや公式ページの成分表示（テキスト）をコピー
  2. ツールに貼り付ける
  3. 表示された説明を見て、気になる成分を拾う

コツ：まずは上位に出てくる成分ほど配合量が多い傾向があります。

注意（できないこと・限界）

  • 医療・薬機法上の判断や、肌トラブルの診断を行うツールではありません。
  • 成分の「良し悪し」は個人差があります。気になる場合はパッチテストや専門家の相談を推奨します。
  • 表記ゆれ（別名・略称）によっては、説明が出ない成分がある場合があります。
