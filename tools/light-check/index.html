使い方 How to use
  1. 「開始」を押すと、カメラが起動します。
  2. 画面の指標（明るさ/色/安定度など）を確認します。
  3. 必要に応じて「カメラ切替」や「Lite」を利用できます。
  4. 終了時は「停止」で停止できます。
  1. Starting the camera is available from “Start”.
  2. The indicators show brightness / color / stability, etc.
  3. Camera switching (“Flip”) and “Lite” mode are available if needed.
  4. Stopping is available from “Stop”.

※ ブラウザの権限設定でカメラが許可されている必要があります。 Camera permission is required in your browser settings.

見方のヒント Reading tips
  • 明るさが低い場合：ライトを近づける／ISOを下げる等の対策が有効です。
  • 色かぶりがある場合：照明の色温度を揃える、混色を避けると安定します。
  • ちらつきが疑われる場合：周波数の異なる照明や調光は影響要因になり得ます。
  • If it’s too dark: moving the light closer or reducing ISO may help.
  • If there’s a color cast: matching color temperatures and reducing mixed lighting may help.
  • If flicker is suspected: dimming or mixed-frequency light sources can be contributing factors.
制限事項 Limitations
  • 本ツールは簡易計測です。照度計（lux meter）の代替ではありません。
  • フリッカー検出は端末のFPSや露出制御の影響を受けます。
  • 端末/ブラウザによってカメラ切替や取得できる情報が異なる場合があります。
  • This is a lightweight measurement tool and not a replacement for a lux meter.
  • Flicker detection depends on FPS and camera exposure behavior.
  • Available camera features vary by device and browser.
