Contract Risk Highlighter の使い方

契約書テキストを貼り付けて、解除、損害賠償、免責、支払、検収、権利帰属など、注意が必要になり得る条項パターンを確認する無料ツールです。

このツールは法的助言ではありません。 契約可否、法的効果、条文修正の正しさは保証しません。重要な契約は弁護士・専門家・社内法務に確認してください。

使い方

  1. 契約書本文をテキストとして貼り付けます。
  2. 契約タイプを「業務委託」「NDA」「売買」から選びます。
  3. Analyze を押します。
  4. Risk level、Findings、確認質問、Markdown preview を確認します。
  5. Copy markdown、Download .md、Print / Save PDF のいずれかで結果を保存・共有します。

対応している確認項目

  • 解除、違約金、無催告解除
  • 損害賠償、免責、間接損害、逸失利益
  • 支払条件、検収、みなし承認、遅延損害金
  • 成果物の権利帰属、著作者人格権不行使、ノウハウ譲渡
  • 秘密保持、競業避止、専属契約、再委託、個人情報
  • 準拠法・管轄、反社条項の不足確認

非対応・注意点

  • PDF本文の直接抽出は準備中です。現在はテキスト貼り付けのみ対応です。
  • 解析結果はブラウザの印刷機能でPDF保存できます。
  • 検出は正規表現ベースの簡易チェックです。検出漏れや誤検出があります。
  • 検出されない場合でも、契約書が安全であることを意味しません。
  • 機密契約書、個人情報、未公開情報は貼り付けないでください。
  • 本文解析はブラウザ内で行います。ただしページ表示のために広告・解析タグは読み込まれます。

出力について

Markdown preview は無料で利用できます。原文全文は含めず、検出箇所の短い抜粋、確認理由、確認質問、修正・確認の方向性だけを出します。

Download .md でMarkdownファイルとして保存できます。Print / Save PDF はブラウザの印刷画面を開きます。PDFとして保存する場合は、印刷画面で「PDFに保存」を選んでください。

FAQ

法的助言ですか？

いいえ。注意パターンを機械的に拾う簡易チェックです。

契約書テキストは送信されますか？

本文解析はブラウザ内で行います。ただし広告・解析タグは読み込まれます。

PDFに対応していますか？

PDF本文の直接抽出は準備中です。解析結果はブラウザの印刷機能でPDF保存できます。
