Contract Risk Highlighter の使い方

契約書テキストを貼り付けて、解除、損害賠償、免責、支払、検収、権利帰属、秘密保持、再委託、個人情報などの注意候補をブラウザ内で確認するツールです。

これは法的助言ではありません。 契約可否、法的効果、条文修正の正しさは保証しません。重要な契約は弁護士・専門家・社内法務に確認してください。

無料版でできること

  • 契約書テキスト貼り付け
  • 契約タイプ選択（業務委託 / NDA / 売買）
  • Analyze、Risk表示、主要Findings最大3件
  • Lite Markdown preview と Lite Markdown copy
  • JP/EN切替、Paste example、FAQ確認

NicheWorks Proでできること

  • Findings全件表示
  • Full Review Markdown
  • .md download
  • Print / Save PDF（ブラウザ印刷画面でPDF保存）
  • 専門家相談メモ、相手方への確認質問リスト、Missing Clause Checklist、Next Action Memo
  • Pro専用copy/export

購入後の扱い

NicheWorks Proはツール個別Proではなく共通Proです。購入後、/pro/unlock/?session_id=... で有効化され、このブラウザではNicheWorks Proが有効になります。タブやブラウザを閉じても通常は維持されます。ただし、別端末・別ブラウザ・シークレットモード・サイトデータ削除後は再度有効化が必要です。localStorageは補助状態であり、サーバー側の pro_entitlements が正です。

非対応・注意点

  • PDF本文の直接抽出は整備済みです。現在はテキスト貼り付けのみ対応です。
  • ProのPDFは解析結果をブラウザの印刷機能で保存するものです。
  • 検出はルールベースの簡易チェックです。検出漏れや誤検出があります。
  • 検出されない場合でも、契約書が安全であることを意味しません。
  • 機密契約書、個人情報、未公開情報は貼り付けないでください。
  • 本文解析はブラウザ内で行います。ただし広告・解析タグは読み込まれます。

FAQ

法的助言ですか？

いいえ。注意パターンを機械的に拾う簡易チェックです。

契約書テキストは送信されますか？

本文解析はブラウザ内で行います。ただし広告・解析タグは読み込まれます。

PDFに対応していますか？

PDF本文の直接抽出は整備済みです。解析結果はブラウザの印刷機能でPDF保存できます。
