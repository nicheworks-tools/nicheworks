Not legal advice. This is a checklist that flags common patterns and suggests questions to confirm.

Choosing a mode

  • JP mode: Japanese contracts / Japanese practice check
  • UK mode: English contracts with an English-law-oriented checklist

Steps

  1. Open the tool
  2. Select Mode (JP / UK)
  3. Paste the contract text
  4. Run analysis
  5. Use the suggested questions to confirm critical points
  6. Export reports with Pro if needed
