Vibe Lexicon の使い方

曖昧な依頼から実務向けの言葉へ素早く変換するための使い方をまとめています。現行公開版は無料で、検索・詳細・比較・コピーを中心に利用できます。

基本ワークフロー

Step 1

意図語で検索する

modern / readable / premium / clean など、まずはざっくりした語から開始します。候補表示で近い用語へすぐ移動できます。

Step 2

フィルタで目的を絞る

カテゴリ・用途で絞ると、似て見える語の使い分けが明確になります。見た目改善と情報整理の混同を防ぐのに有効です。

Step 3

詳細で意味と実務意図を確認

平易説明と分解表示（曖昧→具体）を確認し、実装・レビューで使える依頼文へ書き換えます。

Step 4

近接語を比較する

clean と minimal、premium と refined のような近い語を比較し、求めるUI結果に最も合う表現を選びます。

お気に入り・最近見た項目

現時点ではブラウザのローカル保存です。同じ端末・同じブラウザでの再利用には便利ですが、端末間同期やアカウント連携は未対応です。

無料版で現在できること

  • 検索・フィルタ・用語参照。
  • 詳細確認と実務向け言い換えの把握。
  • 基本比較。
  • 短いプロンプト文のコピー。
  • ローカル保存のお気に入り・最近見た項目。

現行公開版の範囲と注意点

  • アカウント機能・クラウド同期は未対応。
  • 無料版でも中核機能は実用レベルで使える。
  • 未実装の有料ロックUIを見せる構成にはしていない。

機能拡張の方向性は別ページで透明性を持って公開しています。
