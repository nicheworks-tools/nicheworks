Vibe Lexicon について

Vibe Lexicon は「なんとなくこうしたい」を、設計と実装に使える言葉へ変換するための表現判断ツールです。曖昧な依頼を、実務で伝わる指示文に近づけます。

曖昧な依頼 → 実務で使える表現 UI改善の言語化 現場で再利用できる基礎辞書

このツールは何をするのか

Vibe Lexicon は単なる用語辞典ではありません。たとえば「モダンにしたい」「読みやすくしたい」「CTAをわかりやすくしたい」といった曖昧な要望を、情報設計やUI実装で使える実践的な言葉へ落とし込むための参照ツールです。

誰に向いているか

  • 抽象的な要望を具体化してからデザインしたい人。
  • レビュー指示を実装可能な文に整えたい開発者。
  • 「高級感」「洗練」「可読性」などの言葉をチームで揃えたい担当者。
  • 短いAIプロンプトを現実的なUI改善に使いたい人。

解決したい表現のズレ

ありがちな依頼

「今っぽくして」「フォームを改善して」「モバイルを見やすくして」

実務向けの言い換え

「余白リズムを整え、補助ラベルを整理し、CTAの視認性を上げて初見スキャン性を改善する」

主な対象は modern / polished / premium / readable / CTA clarity / clutter reduction / mobile readability / form clarity です。

一般的な用語集やプロンプト集との違い

意味説明だけで終わらず、実務での使いどころ・悪い依頼と良い依頼の差・短い再利用文までを一体で確認できます。雰囲気語をそのまま投げるのではなく、具体的な設計意図へ変換することを重視しています。

現在の無料版でできること

  • 用語の検索・フィルタ・参照。
  • 詳細での平易説明と分解表示。
  • 基本的な比較表示。
  • 短いプロンプト文のコピー。
  • お気に入り・最近見た項目（ブラウザローカル保存）。

今後の方向

将来的には、より深い比較や継続利用向けの機能を Pro 側で拡張予定です。現時点での方針は Proページ で公開しています。
