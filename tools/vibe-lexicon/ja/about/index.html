Vibe Lexicon について

Vibe Lexicon は、曖昧な依頼を実務で使える設計言語に変換するための表現判断ツールです。「なんとなく良くしたい」を、実装とレビューで共有できる具体表現へ近づけます。

曖昧な依頼 → 実務で使える表現 デザイン・実装・レビューの共通言語 現場で再利用できる判断基盤

このツールの役割

Vibe Lexicon は、modern / polished / premium のような雰囲気語を、CTAの明瞭化・可読性改善・フォーム説明・情報整理といった実行可能な言葉へ落とし込むための参照レイヤーです。UIの「方向性はあるが言語化が弱い」状態を減らすことを目的にしています。

向いている人

  • 抽象レビューを具体に変換してから設計したい人。
  • 曖昧な修正依頼を実装タスクへ落としたい開発者。
  • チーム内の語彙差を減らしたいPM・運用担当。

用語集・プロンプト集との違い

意味説明で終わらない

単語の定義だけでなく、実務意図と依頼文の言い換えに重心を置いています。抽象語をそのまま投げず、意思決定しやすい形に整えます。

雰囲気だけのAI用途に寄せない

短いプロンプト例は、実際のUI改善を前提にした内容です。見た目の流行語収集ではなく、改善実装の会話を進めるための設計です。

Vibe Lexicon は「言葉の辞典」ではなく「判断の辞典」です。

現在の無料版でできること

  • 公開語彙の検索・フィルタ・参照。
  • 詳細説明と分解表示（曖昧→具体）。
  • 近接語の基本比較。
  • 短いプロンプト文のコピー。
  • お気に入り・最近見た項目（ブラウザローカル保存）。

今後の方向

無料版は実用基盤として維持しつつ、Pro側では深い比較と再利用性（保存セット等）を段階的に拡張する方針です。未実装機能を「すでに使える」ようには見せません。
