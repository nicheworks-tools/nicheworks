最短の使い方

Fastest path

  1. 検索欄に「今っぽい」「ごちゃついている」「CTAが弱い」など、悩みのまま入力します。
  2. 該当語を開き、「曖昧語からの分解」を見て、何を直す話なのかを理解します。
  3. 用途別タブで UI / LP / フォーム / モバイル の依頼文を切り替えます。
  4. 短文コピーでAIへ投げ、必要なら比較トレイで近い言葉との差を見ます。
  1. Type the problem as-is: “make it modern,” “feels cluttered,” “CTA is weak,” and so on.
  2. Open a term and read the breakdown to understand what the request usually means in practice.
  3. Switch between UI / LP / form / mobile prompt modes.
  4. Copy the short prompt, then compare nearby terms when needed.

何を見るべきか

What to pay attention to

ひらたく言うと Plain explanation

その言葉が、結局どんな変更を求めているのかをざっくり把握する場所です。

A quick explanation of what the term usually asks you to change.

曖昧語からの分解 Breakdown

「高級感」や「今っぽい」を、余白・色数・CTA数など具体要素へ分解します。

Turns vague terms into concrete changes such as spacing, color restraint, or CTA reduction.

悪い依頼例 / 良い依頼例 Bad vs better requests

曖昧なまま投げるとズレやすいので、どこまで具体化すればよいかを見ます。

Shows how far you need to clarify a request before sending it to AI.

用途別依頼文 Use-case prompts

同じ言葉でも UI、LP、フォーム、モバイルで使い方が違うため、用途別に切り替えます。

The same term behaves differently in UI, LP, forms, and mobile, so switch prompt modes accordingly.

比較が役立つ場面

When compare helps

  • 「今っぽい」と「洗練された」の違いを言語化したい時
  • 「見やすい」「読みやすい」「分かりやすい」のどれを使うべきか迷う時
  • 同じ悩みを別の言葉で依頼した方が通りそうか確認したい時
  • When you need to tell “modern” apart from “refined.”
  • When you’re unsure whether “readable,” “clear,” or “easy to scan” is the right request.
  • When you want to test whether a neighboring term will produce better AI output.

無料とProの違い

Free vs Pro

無料では、意味確認・検索・基本比較・短文コピーができます。Proでは、用途別文例のフル解放、3語以上の比較、保存、履歴、語群まとめコピーを想定しています。

The free tier covers search, explanation, basic compare, and short-copy actions. Pro unlocks full prompt variants, 3+ compare, save, history, and bundle copy.
