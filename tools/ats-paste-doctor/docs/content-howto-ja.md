# 結論
応募フォーム/ATSに貼り付けたときの「改行崩れ・箇条書き崩れ・変な空白」は、貼る前にテキストを整えると回避できる場合があります。
ただし、ATS側の仕様で完全には防げないことがあります。

# よくある症状
- “paste line breaks removed” のように改行が消える
- 箇条書き（• - *）が崩れて1行になる
- 余計な空白やタブで見た目がズレる
- 文字数制限で “too long” と出る
- 原因不明の入力エラー（不可視文字が混ざっている）

# 使い方（最短3ステップ）
1. ATS Paste Doctor を開いてテキストを貼り付ける
2. 「ATS安全」を選んで「出力を作る」
3. 警告と2行プレビューで崩れ方を確認し、出力をコピーして貼る

# 注意点
- ATS/フォーム側の仕様で、改行や記号が強制変換されることがあります。
- 機密情報は貼り付けないことを推奨します（ブラウザ内処理ですが、自己責任で）。
