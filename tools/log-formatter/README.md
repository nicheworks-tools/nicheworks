# 地味ログ整形屋（LogFormatter）

Nginxアクセスログなどのテキストログを、ブラウザ内だけで整形・色分け・フィルタ表示するツールです。

- 公開URL: https://log-formatter.pages.dev/
- ログはすべてブラウザ内で処理され、サーバーには送信されません。

## 使い方（チェック用）

1. 上部テキストエリアに Nginx アクセスログをペーストします。
2. 「整形して表示」を押します。
3. 下部に行ごとの整形結果が表示されます。
   - IP / 時刻 / メソッド / URL / ステータス / サイズ が色分けされます。
4. ステータスフィルタや文字列フィルタを使うと、
   条件に合う行だけを表示できます。

## 動作確認チェックリスト

- [ ] ログを貼ると行ごとに表示される
- [ ] 200系が緑、404など4xxがオレンジ、500系が赤になる
- [ ] 「含む文字列」で `login` と入れると login を含む行だけになる
- [ ] 「除外文字列」で `healthcheck` と入れると healthcheck 行が消える
- [ ] 「条件に合う行のみ表示」をONにするとノイズ行が消える
