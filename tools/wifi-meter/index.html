WiFi Meter｜Connection Estimate

Network Information APIが返す推定RTT・推定帯域をもとに、通信状態の目安を表示します。

Shows a connection estimate based on RTT and downlink values reported by the Network Information API.

WiFi診断・速度測定・ping診断の正確な代替ではありません。

This is not a precise replacement for WiFi diagnostics, speed tests, or ping tools.

FAQ

FAQ

Q. WiFiのSSIDやパスワードを取得しますか？
いいえ。SSID、パスワード、接続先一覧、周辺WiFi一覧にはアクセスしません。

Q. Does this read WiFi SSIDs or passwords?
No. It does not access SSIDs, passwords, connected-network lists, or nearby WiFi lists.

Q. 正確な速度測定ですか？
いいえ。Network Information APIが返す推定RTT・推定帯域の目安です。速度測定サイトやping診断の代替ではありません。

Q. Is this an accurate speed test?
No. It shows estimated RTT and downlink values from the Network Information API, not a replacement for speed tests or ping diagnostics.

Q. どのブラウザでも使えますか？
いいえ。一部ブラウザではNetwork Information APIが使えず、値が表示されない場合があります。

Q. Does it work in every browser?
No. Some browsers do not support the Network Information API, so values may not be shown.

Q. ページ上で外部通信はありますか？
ツールの入力値は送信しませんが、ページ表示のためGA4、AdSense、Cloudflare Analyticsなどが読み込まれます。

Q. Does the page make external requests?
The tool does not upload user-entered data, but GA4, AdSense, Cloudflare Analytics, and similar page scripts may load.
