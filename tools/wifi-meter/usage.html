How to Use｜WiFi Meter

このページでは、ツールの使い方、取得値の意味、対応ブラウザ差、免責について説明します。

This page explains how to use the tool, what each value means, browser support differences, and disclaimers.

1. このツールでできること

1. What This Tool Does

  • Network Information APIが返す推定RTT・推定帯域を表示します。
  • Shows estimated RTT and downlink values returned by the Network Information API.
  • RTTの変動幅から通信状態の目安（低負荷・中程度・高負荷）を表示します。
  • Uses RTT fluctuation to show a connection estimate: low, medium, or high load.
  • 短いRTTトレンドグラフを表示します。
  • Shows a short RTT trend graph.
  • WiFiの電波強度、チャンネル混雑、接続台数、ルーター状態は測定できません。
  • It cannot measure WiFi signal strength, channel congestion, connected device count, or router status.

2. 使い方

2. How to Use

  1. 「Start」を押します。
  2. Press “Start”.
  3. RTT、変動幅、推定帯域、通信状態の目安を確認します。
  4. Check RTT, fluctuation, estimated downlink, and the connection estimate.
  5. 確認が終わったら「Stop」を押します。
  6. Press “Stop” when finished.
  7. 結果を消す場合は「Reset」を2回押します。
  8. Press “Reset” twice to clear the result.

3. 対応 / 非対応

3. Supported / Unsupported

対応

Supported

  • Network Information APIに対応したブラウザでの推定RTT表示
  • Estimated RTT display in browsers that support the Network Information API
  • 対応ブラウザでの推定downlink表示
  • Estimated downlink display in supported browsers
  • ページ内での簡易グラフ表示
  • Simple in-page trend graph

非対応

Unsupported

  • WiFiのSSID、パスワード、接続先一覧、周辺WiFi一覧の取得
  • Reading WiFi SSIDs, passwords, connected-network lists, or nearby WiFi lists
  • WiFi電波強度、チャンネル混雑、ルーター状態、接続台数の測定
  • Measuring WiFi signal strength, channel congestion, router status, or connected device count
  • 速度測定サイトやping診断と同等の正確な測定
  • Accurate measurement equivalent to speed-test or ping-diagnostic services
  • Safari/iOSなど、Network Information APIが利用できないブラウザでの値表示
  • Showing values in browsers such as Safari/iOS where the Network Information API may be unavailable

4. データ取得と免責

4. Data Acquisition & Disclaimer

取得する可能性がある値

Values This Tool May Read

  • rtt：ブラウザが返す推定RTTです。実測pingとは異なる場合があります。Browser-reported estimated RTT. It may differ from an actual ping measurement.
  • downlink：ブラウザが返す推定下り帯域です。速度測定サイトの実測値ではありません。Browser-reported estimated downlink. It is not the same as a measured speed-test result.
  • 変動幅 / fluctuation：前回RTTとの差分をツール内で計算した目安です。A local estimate calculated from the difference between current and previous RTT values.

取得しない情報

Data This Tool Does Not Access

  • WiFiのSSID、パスワード、周辺WiFi一覧、接続先一覧
  • WiFi SSIDs, passwords, nearby WiFi lists, or connected-network lists
  • 通信内容、個人ファイル、位置情報、ルーター管理情報
  • Communication contents, personal files, location data, or router admin information
  • 任意のWiFiへ接続する操作
  • Connecting the device to a WiFi network

外部タグについて

External Page Scripts

ツールの入力値や測定値を外部へ送信する処理は実装していません。ただし、ページ表示のためGA4、AdSense、Cloudflare Analyticsなどの広告/解析タグが読み込まれます。

The tool does not implement uploading user-entered or measured values. However, GA4, AdSense, Cloudflare Analytics, and similar ad/analytics scripts may load for the page.

免責

Disclaimer

本ツールはブラウザ推定値を使った簡易表示です。実際の通信速度、遅延、WiFi電波状況、ルーター状態を保証するものではありません。重要な診断や契約判断には、公式の速度測定、ルーター管理画面、通信事業者の情報などを確認してください。

This tool is a simple display based on browser-estimated values. It does not guarantee actual speed, latency, WiFi signal conditions, or router status. For important diagnostics or contract decisions, check official speed tests, router admin tools, or your network provider.

5. よくある質問

5. FAQ

Q. WiFiのSSIDやパスワードを取得しますか？
いいえ。SSID、パスワード、周辺WiFi一覧にはアクセスしません。

Q. Does this read WiFi SSIDs or passwords?
No. It does not access SSIDs, passwords, or nearby WiFi lists.

Q. 正確な速度測定ですか？
いいえ。Network Information APIが返す推定値の目安です。

Q. Is this an accurate speed test?
No. It is an estimate based on values returned by the Network Information API.

Q. SafariやiPhoneで使えますか？
Network Information APIが利用できない場合、値は表示されません。

Q. Does it work on Safari or iPhone?
If the Network Information API is unavailable, values will not be displayed.
