使い方｜WiFi 混雑度チェッカー

How to Use｜WiFi Congestion Meter

このページでは、ツールの使い方と取得データの意味、安全性について説明します。

This page explains how to use the tool, what each value means, and why it is safe.

1. このツールでできること

1. What This Tool Does

  • ブラウザが返す推定値からWiFi混雑度を表示
  • Shows estimated WiFi congestion using browser-provided values
  • 混雑レベル（低・中・高）を可視化
  • Displays congestion levels (Low / Medium / High)
  • RTT（推定遅延）・変動幅・帯域推定を簡易表示
  • Displays estimated RTT, fluctuation, and bandwidth
  • リアルタイムのミニ折れ線グラフ
  • Mini real-time line graph
  • 完全ローカル動作（接続や個人情報取得なし）
  • Runs fully locally (no connection or personal data access)

2. 使い方

2. How to Use

  1. WiFiをONにする（接続は不要）
  2. Turn on WiFi (connection not required)
  3. 「Start」を押す
  4. Press “Start”
  5. 混雑レベル・数値・グラフを確認
  6. Check congestion level, values, and graph
  7. 必要に応じて「Stop」→「Reset」
  8. Press “Stop” then “Reset” if needed

3. 取得するデータとその意味

3. Data Obtained & Meaning

▼ 専門説明

▼ Technical Explanation

  • RTT（推定遅延 / ms）
    ブラウザが推定したネットワーク遅延。混雑・干渉・電波弱化で増加しやすい。 Estimated network latency. Increases with congestion, interference, or weak signal.
  • downlink（推定帯域 / Mbps）
    ブラウザが推定する下り帯域。実測値ではなく概算。 Estimated download bandwidth. Not an actual measurement.
  • effectiveType（通信品質カテゴリ）
    通信品質を slow-2g / 2g / 3g / 4g / wifi で分類。 Categorizes network quality as slow-2g / 2g / 3g / 4g / wifi.
  • 変動幅（ツール内部算出）
    短期のRTT揺れ幅。大きいほどネットが不安定。 Short-term RTT variation. Larger = more unstable network.

▼ 初心者向け “つまりどういうこと？”

▼ Beginner-Friendly Summary

  • RTTが高いほど混雑の可能性が高い
  • Higher RTT = likely congestion
  • 変動幅が大きいほど不安定
  • Large fluctuation = unstable
  • 推定帯域が低い＝混雑 or 電波弱化
  • Very low bandwidth = congestion or weak signal
  • 数値はすべて“目安”で、精密計測ではない
  • All values are approximate, not precise measurements

4. データ取得と免責（最重要）

4. Data Acquisition & Disclaimer (Important)

▼ 取得するデータ（すべて推定）

▼ Data This Tool Reads (estimated only)

  • RTT（推定遅延）
  • downlink（推定帯域）
  • effectiveType（品質カテゴリ）
  • 揺らぎ（内部計算）

▼ 技術的に取得不可能なデータ

▼ Data That Cannot Be Accessed

  • 周囲のWiFi一覧（SSID）
  • パスワード
  • 暗号化方式
  • ルーター情報
  • 個人情報
  • 通信内容
  • WiFi接続操作（JavaScriptでは実行不可）
  • GPS位置情報（許可なしでは不可）

▼ 安全性の根拠

▼ Why This Tool Is Safe

  • ブラウザAPIは取得範囲が厳しく制限されている
  • Browser APIs strictly limit accessible information
  • 任意のWiFiへ接続することは技術的に不可能
  • JavaScript cannot initiate WiFi connections
  • データ送信は行わず、端末内のみで処理
  • All processing is done locally with no data uploaded

▼ 免責

▼ Disclaimer

本ツールは推定値を使用した簡易診断であり、実際の通信速度や品質を保証するものではありません。

This tool provides approximate values only and does not guarantee actual network performance.
