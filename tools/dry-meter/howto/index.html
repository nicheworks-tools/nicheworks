DryMeter：使い方

雨・湿度・風・気温から、外干しの「失敗しやすさ」を🟢🟡🔴で判断します（保存なし）。

広告枠

このツールでできること

  • 現在地の天気（雨 / 湿度 / 風 / 気温）を元に、外干しリスクを 🟢🟡🔴 で表示
  • 洗濯物 / 布団 の切替がある場合は、用途に合わせて判定を変える
  • 結果の理由を短く表示（例：湿度が高い / 風が弱い / 雨リスク 等）
  • ブラウザ完結・保存なし
ツールを開く English guide

こんなときに使う

  • 「今日、外に干して大丈夫？」を即判断したい
  • 布団を干すか迷っていて、失敗（戻す羽目）を減らしたい
  • 雨は降らなさそうだが、湿度や風が微妙で乾くか不安

関連キーワード：外干し 失敗 / 洗濯 乾きやすい日 / 布団 外干し 天気 / humidity wind dry forecast

使い方

  1. ツールを開き、位置情報の利用を許可（現在地の天気取得のため）
  2. 洗濯物 / 布団 が選べる場合は用途を選ぶ
  3. 表示された判定（🟢🟡🔴）と理由を見て、外干し可否を決める

コツ：🟡の日は「短時間だけ干す」「風通しを作る」「浴室乾燥に切替」などで事故回避しやすい。

注意（できないこと・限界）

  • 天気APIの誤差や、局地的な雨・日陰・建物の風の弱さなどは反映しきれない場合があります。
  • 🟢でも突然の雨はゼロではありません。心配なら取り込み時間を早めに設定してください。
  • 位置情報を許可しない場合、判定ができない/精度が落ちる可能性があります。
