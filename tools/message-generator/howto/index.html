このツールでできること

  • 即生成依頼 / お礼 / 断り / 催促 / 謝罪 / 日程調整 などの短文テンプレを作る
  • トーン丁寧・普通・カジュアル（ツール側の選択肢に依存）で言い回しを切り替える
  • 用途ビジネス / 日常 / SNS の「ちょうどいい長さ」を狙う
  • ブラウザ完結・保存なし
ツールを開く English guide

よくある用途

  • 返信が遅れたときの一言（謝罪＋要点）
  • お願い・依頼を「角が立たない」言い方にしたい
  • 断りたいけど、感じ悪くしたくない
  • 催促したいけど、圧を下げたい

関連キーワード：文面 テンプレ / 断り方 例文 / お礼 メッセージ / 催促 文章 / business message template

使い方（最短3ステップ）

  1. 用途（依頼/お礼/断り/催促…）と、相手（上司/取引先/友人…）を選ぶ
  2. 必要な情報（用件・期限・お礼など）を短く入力
  3. 出力された文面を少し直してコピーして送る

コツ：固有名詞と日付だけ自分で入れると、事故率が一気に下がる。

注意（できないこと・限界）

  • 相手との関係性や社内ルールによって、適切な敬語レベルは変わります。
  • 法務・契約・謝罪文など重要な文章は、そのまま送らず必ず見直してください。
  • 個人情報や機密情報は入力しないでください。
