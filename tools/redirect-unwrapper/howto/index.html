Redirect Unwrapper：使い方

短縮URLの「本当の行き先」を見てから踏む。

広告枠（準備中）

このツールでできること

  • 解析短縮URL / SNSリンク / 301302 を追跡して最終URLを表示
  • 安全怪しいリンクの踏み抜き防止（共有前チェックにも使える）
  • 用途フィッシング対策、広告リンク確認、リダイレクト事故防止
ツールを開く English guide

よくある症状

  • 短縮URLで行き先が見えず不安（bit.ly / t.co など）
  • SNSの「外部リンク」が別サイトを経由していて分かりづらい
  • 共有されたURLが怪しく見える（踏む前に確認したい）
  • 最終URLが違うドメインに飛ぶ（広告/追跡/フィッシングの可能性）

関連キーワード：短縮URL 解析 / 最終URL 確認 / リダイレクト 追跡 / t.co 行き先 / フィッシング URL

使い方（最短3ステップ）

  1. 短縮URL（または怪しいURL）を貼り付ける
  2. 解析ボタンを押す（または自動解析が走る）
  3. 表示された最終URL（ドメイン）を確認してから開く/共有する

コツ：最終URLだけでなく「途中経路」が出るなら、途中に見慣れない追跡ドメインがないかも見る。

注意（できないこと・限界）

  • サイト側のブロックやCORS等により、すべてのリンクが解析できるとは限りません。
  • 最終URLが安全である保証はできません（判断材料の提示ツールです）。
  • ログイン必須ページ等は最終到達先が変わる場合があります。
