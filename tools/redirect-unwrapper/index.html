This tool analyzes the URL string locally in your browser without opening it. It does not fully follow server-side 301/302 redirects from short URLs.

      FAQ

      このツールはURLを開きますか？Does this tool open the URL?

      いいえ。URL文字列を解析し、埋め込まれた転送先候補を抽出します。No. It analyzes the URL string and extracts embedded destination candidates.

      bit.ly や t.co の最終URLを完全に追跡できますか？Can it fully resolve bit.ly or t.co final URLs?

      いいえ。サーバー側リダイレクトの完全追跡は行いません。No. It does not follow server-side redirects.

      表示された候補URLは安全ですか？Are extracted candidates safe?

      安全性は保証しません。不明なリンクは開かないでください。Safety is not guaranteed. Do not open unknown links.

      URLはサーバーに送信されますか？Is the URL sent to a server?

      入力URLの解析はブラウザ内で行います。ただしページ表示のため広告・解析タグは読み込まれます。The entered URL is analyzed in the browser. Ads and analytics scripts may still load for the page itself.
