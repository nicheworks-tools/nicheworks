Redirect Unwrapper の使い方

Redirect Unwrapper は、追跡リンクやSNSリンクのURL文字列に含まれる転送先URL候補を確認するための無料ツールです。

広告枠
重要：このツールは入力URLを開きません。外部サイトへアクセスせず、ブラウザ内でURL文字列だけを解析します。短縮URLのサーバー側301/302リダイレクトを完全追跡するものではありません。

1. このツールでできること

  • URLのスキーム、ホスト、パスを分解して表示します。
  • url、u、q、redirect_url、destination などのパラメータ内にあるURL候補を抽出します。
  • 二重エンコードされたURL候補も可能な範囲で展開します。
  • UTM、fbclid、gclid などの追跡パラメータを表示します。
  • HTTP、危険なスキーム、punycode、IPアドレスhost、表示hostと候補hostの不一致などを警告します。

2. 使い方

  1. 解析したいURLを入力欄に貼り付けます。
  2. 「解析する」を押します。
  3. 「URL分解」「抽出された転送先候補」「警告・確認ポイント」を確認します。
  4. 必要なら候補URLまたは解析結果サマリーをコピーします。

3. 対応している入力例

https://www.google.com/url?q=https%3A%2F%2Fexample.com%2F

https://mail.example/click?redirect_url=https%253A%252F%252Fexample.com

https://example.com/page?utm_source=x&utm_medium=social

4. 非対応・限界

  • bit.ly、t.co、amzn.to などのサーバー側リダイレクトは追跡しません。
  • 候補URLが表示されても、安全性は保証しません。
  • URL短縮サービスの実際の到達先確認には、別途安全な環境や公式の確認手段が必要です。
  • このページ自体の表示に必要な広告・解析タグは読み込まれます。

5. 警告の見方

  • HTTP URL：通信が暗号化されていない可能性があります。
  • javascript / data / file：通常のWebリンクとして扱うべきではありません。
  • punycode：見た目が紛らわしいドメインの可能性があります。
  • IPアドレスhost：一時URL、管理画面、偽装リンクの可能性に注意してください。
  • host mismatch：見えているドメインと転送先候補が異なります。

6. よくある質問

このツールはURLを開きますか？

いいえ。URL文字列を解析するだけです。

表示された候補URLは安全ですか？

安全性は保証しません。不明なリンクは開かないでください。

URLはサーバーに送信されますか？

入力URLの解析はブラウザ内で行います。ただしページ表示のため広告・解析タグは読み込まれます。

7. 免責

本ツールはURL文字列の確認補助です。リンク先の安全性、正確性、到達先を保証するものではありません。重要な確認は公式情報や安全な検証環境で行ってください。
