How to use

  1. Look at the care label on your clothing
  2. Tap the matching icon in the tool
  3. Read the meaning (allowed / not allowed / temperature / notes)

Tip: check in this order: wash → bleach → dry → iron → dry clean.

Notes (limits)

  • Fabric type matters (wool/silk etc.). Use symbols as guidance plus common sense.
  • The care label is the final source of truth.
  • Some brands may use non-standard symbols.
