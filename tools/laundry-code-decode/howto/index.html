Laundry Code Decode：使い方

衣類タグの「洗濯記号」を選ぶだけで、意味と注意点を即表示します。

広告枠（準備中）

このツールでできること

  • 洗濯記号（国際表記）の意味をすぐ確認
  • 洗濯・漂白・乾燥（タンブル/自然乾燥）・アイロン・ドライの可否を整理
  • 日本語 / 英語 表示に対応（ツール内切替）
  • 完全ローカル動作・保存なし
ツールを開く English guide

こんなときに使う

  • 「たらいに×」とか「三角に×」の意味が分からない
  • 乾燥機OKか、アイロン温度は何度かを知りたい
  • 海外ブランドの服で、英語表記でも確認したい

関連キーワード：洗濯記号 意味 / ランドリーシンボル 解説 / dryer symbol meaning / laundry symbols guide

使い方

  1. 服のタグにある記号を見て、同じアイコンをツールでタップ
  2. 表示された意味（OK/NG・温度・注意点）を確認
  3. 必要なら言語を切り替えて（JA/EN）もう一度確認

コツ：複数の記号が並んでいる場合は、洗濯→漂白→乾燥→アイロン→ドライの順に確認すると迷わない。

注意（できないこと・限界）

  • 同じ記号でも、素材（ウール/シルク等）で実際の難易度は変わります。
  • タグ表記が最優先です。迷ったら「弱水流・ネット・陰干し」など安全側で。
  • 一部ブランド独自表記は、国際記号と一致しない場合があります。
