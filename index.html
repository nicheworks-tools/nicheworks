無料ブラウザツール集

NicheWorks は、PDF、画像、CSV、JSON、文章整形、開発補助などの小さな作業を、インストールなしで処理するためのWebツール集です。多くのツールはブラウザ内で処理し、入力内容をサーバーに保存しません。

主要ツール

検索エンジンと利用者の両方がたどれるよう、主要ツールへのリンクをHTML内にも固定表示しています。

  • 旧字体一覧・旧字新字対応表
    旧字体と新字体の対応を検索・確認できるリファレンス。
  • JSON Repair
    壊れたJSONを修正候補つきで整えるブラウザ完結ツール。
  • JSON to Mermaid
    JSON構造をMermaid図へ変換して把握しやすくします。
  • PDF to CSV Local
    PDF内の表データをCSV化するローカル処理向けツール。
  • PDF Page Tools Mini
    PDFページの分割・並べ替え・削除などを行う軽量ツール。
  • CSV Tidy
    CSVの列整理、ヘッダ名変更、空白トリムなどの整形を行います。
  • EXIF Cleaner Mini
    画像のEXIFなど共有前に消したいメタデータを確認・削除します。
  • Image Redact
    スクリーンショットや画像の一部を黒塗り・ぼかしで隠します。
  • API Key Token Redactor
    貼り付け前のテキストからAPIキーやトークンらしき値を検出・マスクします。
  • Command Safety Checker
    実行前のコマンドに危険操作が含まれていないか確認します。
  • ManualFinder
    製品名などから公式マニュアル検索を補助します。
  • TrashNavi
    自治体のごみ分別・粗大ごみ情報への到達を補助します。
