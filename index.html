このサイトについて

NicheWorks（ニッチワークス）は、
目立たないけれど確かに必要なものを、
小さく、静かに、かたちにしていくウェブプロジェクト群です。
派手さよりも実用性を、話題性よりも継続性を重視しています。

→ プロジェクトの背景や思想は About ページ にまとめています。

ツール一覧

文章・テキスト系

テキスト処理・ログ解析などの軽量ツール群です。

  • ManualFinder（製品マニュアルリンク集）
    家電・カメラ・スマホなどの公式マニュアルページまとめ。
  • TrashNavi（ごみ分別リンク集）
    自治体ごとの公式ごみ分別ページ・カレンダー。
  • 地味ログ整形屋（LogFormatter）
    Nginx / HTTP アクセスログ整形。ローカル処理。
  • Rename Wizard（ファイル名整形）
    ファイル名の一括整形。プレビューあり。
  • Contract Cleaner（契約書クリーナー）
    契約書テキストの危険語句を可視化。
  • EXIF Cleaner Mini
    画像のEXIF情報を削除（ローカル処理）。
  • LineBreak Doctor（改行修復）
    SNS投稿向けの不要改行修正。
  • URL Title Collector
    複数URLのタイトルを一括取得。
  • Message Generator
    短文テンプレ生成。

画像・ファイル処理

画像・ファイルの変換・解析ツール群です。

  • Image Compression Inspector
    画像形式ごとの圧縮率を比較。
  • Color Replace
    画像内の特定色を別の色へ置換。
  • WebP / AVIF Converter
    画像を軽量形式へ変換。
  • FileType Sniffer
    ファイルの中身を解析して種別判定。

解析・変換ツール

変換・図式化などのユーティリティ群です。

  • JSON → Mermaid 変換
    JSON構造を自動で図に変換。
  • UnitMaster（単位変換）
    多数の単位を高速変換。
  • Kanji Modernizer
    旧字体→新字体変換。
  • Old Kanji Reference
    旧字体一覧辞書。
  • Redirect Unwrapper
    短縮URLの最終遷移先を表示。

生活・実用ツール

日常で役立つ実用ツール群です。

  • JP Postal Lite
    郵便番号→住所検索。
  • 解約どこナビ
    各サブスクの公式解約リンク。

その他ユーティリティ

補助的な軽量ユーティリティ群です。

  • WeatherDiff
    複数API予報の比較。地点入力・現在地対応。
  • WiFi Meter
    周囲Wi-Fiの混雑状況を可視化。
  • Cover Letter Lite
    英文カバーレター生成。
  • Cosmetic Ingredient Checker Lite
    化粧品成分の簡易チェック。

運営方針

多くのツールは「ブラウザ内で完結する」ことを重視しており、 入力したデータはサーバーに送信せずローカルで処理されます。

また、NicheWorks は非営利で運営されていますが、継続のため一部ページに広告を掲載する場合があります。 収益はすべてツールの開発・維持管理・データ整備に利用されます。

応援（寄付）

NicheWorks は個人が作っている小さなツール集です。 気に入ったツールがあれば、コーヒー１杯ぶんだけでも支援いただけるととても助かります。

💌 OFUSE ☕ Ko-fi

寄付は任意です。いただいた支援は開発費・データ整備などに充てています。
