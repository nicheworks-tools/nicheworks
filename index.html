無料ブラウザツール集

NicheWorks は、PDF、画像、CSV、JSON、文章整形、開発補助、プライバシー確認などの小さな作業を、インストールなしで処理するためのWebツール集です。多くのツールはブラウザ内で処理され、入力内容やファイルを当サイトのサーバーへ保存しません。

運営方針

  • 無料で使える小さなツール
    日常作業や開発作業の一部を軽くする、単機能のブラウザツールを公開しています。
  • ローカル処理を優先
    画像、テキスト、CSV、JSONなどは、可能な限りブラウザ内で処理します。外部通信がある場合は各ツール内で説明します。
  • 広告と寄付で運営
    一部ページではGoogle AdSense広告を表示する場合があります。任意の寄付導線も用意しています。

主要ツール一覧

審査・検索エンジン・JavaScript無効環境でも内容が見えるよう、主要ツールを静的HTMLで掲載しています。

PDF・CSV・JSON

  • JSON Repair
    壊れたJSONを修正候補つきで整え、コピーしやすい形にします。
  • JSON to Mermaid
    JSON構造をMermaid図へ変換し、データ構造を把握しやすくします。
  • PDF to CSV Local
    PDF内の表データをCSV化するためのローカル処理向けツールです。
  • PDF Page Tools Mini
    PDFページの分割、並べ替え、削除などを行う軽量ツールです。
  • CSV Tidy
    CSVの列整理、列名変更、空白トリムなどの整形を行います。

画像・ファイル・プライバシー

  • EXIF Cleaner Mini
    写真共有前にEXIFなどのメタデータを確認・削除します。
  • Image Redact
    スクリーンショットや画像の一部を黒塗り・ぼかしで隠します。
  • FileType Sniffer
    ファイルの先頭情報から種類を推定し、共有前の確認を補助します。
  • API Key Token Redactor
    共有前のテキストからAPIキーやトークンらしき値を検出してマスクします。
  • Command Safety Checker
    実行前のコマンドに危険操作が含まれないか確認します。

検索・参照・生活補助

  • ManualFinder
    製品名などから公式マニュアルやサポートページ探しを補助します。
  • 旧字体一覧・旧字新字対応表
    旧字体と新字体の対応を検索・確認できるリファレンスです。
  • 旧字体変換ツール
    旧字体と新字体の比較・変換を補助します。
  • 洗濯表示コード読み解き
    洗濯表示やケアラベルの意味を確認します。
  • UnitMaster
    長さ、重さ、温度などの単位換算を行う無料ツールです。

広告・データの扱い

当サイトでは運営のため広告を表示する場合があります。広告、アクセス解析、Cookie、ツール入力データの扱いは Privacy Policy に記載しています。連絡先は Contact をご確認ください。
