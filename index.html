このサイトについて

NicheWorks（ニッチワークス）は、
目立たないけれど確かに必要なものを、
小さく、静かに、かたちにしていくウェブプロジェクト群です。
派手さよりも実用性を、話題性よりも継続性を重視しています。

→ プロジェクトの背景や思想は About ページ にまとめています。

ツール一覧

文章・テキスト系

  • ManualFinder（製品マニュアルリンク集）
    家電・カメラ・スマホなどの「公式マニュアル／取扱説明書ページ」だけを集めたリンク集。
  • TrashNavi（ごみ分別リンク集）
    自治体ごとの公式ごみ分別ページ・カレンダーへのリンク集。
  • 地味ログ整形屋（LogFormatter）
    Nginx / HTTP アクセスログをブラウザだけで整形・検索するツール。ファイルアップロード不要（ローカル処理）。
  • Rename Wizard（ファイル名整形ツール）
    ファイル名の一括整形用ツール。プレビューしてからコピーでき安全。
  • Contract Cleaner（契約書クリーナー）
    契約書テキストから危険語句をハイライトする下読みチェック用ツール。
  • EXIF Cleaner Mini
    画像ファイルからEXIF情報を削除する軽量ツール。ブラウザ内のみで安全処理。
  • LineBreak Doctor（SNS改行修復ツール）
    X / Instagram / LINE向けの不要改行修正。
  • URL Title Collector（URLタイトル収集ツール）
    複数URLのタイトルを一括取得してメモ化。
  • Message Generator
    用途別の短文テンプレート生成。

画像・ファイル処理

  • Image Compression Inspector（画像圧縮率チェッカー）
    画像形式ごとの圧縮品質を比較。
  • Color Replace（画像色置換ツール）
    画像内の特定色を別の色へ置換。
  • WebP / AVIF Converter
    画像を軽量形式へ変換。
  • FileType Sniffer
    ファイルの中身を解析して種別判定。

解析・変換ツール

  • JSON → Mermaid 変換
    JSON構造を自動で図式化。
  • UnitMaster（単位変換ツール）
    多数の単位を高速変換。
  • Kanji Modernizer
    旧字体 → 新字体へ変換。
  • Old Kanji Reference
    旧字体一覧辞書。
  • Redirect Unwrapper
    短縮URLの最終遷移先を表示。

生活・実用ツール

  • JP Postal Lite（郵便番号検索）
    郵便番号から住所を検索。
  • 解約どこナビ（サービス解約リンク集）
    各種サブスク・サービスの公式解約リンク。

その他ユーティリティ

  • WeatherDiff（天気予報差分チェッカー）
    複数APIの予報比較。地点入力・現在地対応。
  • WiFi Meter（Wi-Fi混雑度チェッカー）
    周囲Wi-Fiの混雑状況を可視化。
  • Cover Letter Lite
    英文カバーレター生成ツール。
  • Cosmetic Ingredient Checker Lite
    化粧品成分の簡易チェック。

運営方針

多くのツールは「ブラウザ内で完結する」ことを重視しており、 入力したデータはサーバーに送信せずローカルで処理されます。

応援（寄付）

NicheWorks は個人が作っている小さなツール集です。 気に入ったツールがあれば、コーヒー１杯ぶんだけでも支援していただけるととても助かります。

💌 OFUSE ☕ Ko-fi

寄付はもちろん完全に任意です。いただいた支援は開発費・データ整備などに充てます。
