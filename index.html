このサイトについて

NicheWorks（ニッチワークス）は、
目立たないけれど確かに必要なものを、
小さく、静かに、かたちにしていくウェブプロジェクト群です。
派手さよりも実用性を、話題性よりも継続性を重視しています。

→ プロジェクトの背景や思想は About ページ にまとめています。

ツール一覧

文章・テキスト系

テキスト整形・ログ解析など、文章処理に特化したツール群です。

  • 地味ログ整形屋（LogFormatter）
    大量ログも高速で整形・検索できる開発者向けツール。ブラウザ完結で安全。
  • ManualFinder（製品マニュアルリンク集）
    家電・カメラ・スマホなどの公式取説リンク集。
  • TrashNavi（ごみ分別リンク集）
    自治体ごとのごみ分別情報まとめ。
  • Rename Wizard
    安全なファイル名の一括整形ツール。
  • Contract Cleaner
    契約書の危険語句を自動ハイライト。
  • EXIF Cleaner Mini
    EXIF情報を完全削除。
  • LineBreak Doctor
    SNS投稿用に不要改行を修正。
  • URL Title Collector
    複数URLのタイトルを取得。
  • Message Generator
    用途別の短文生成。

画像・ファイル処理

画像の変換・圧縮・色処理など、軽量な画像ツール群です。

  • Image Compression Inspector
    形式別の圧縮率を比較。
  • Color Replace
    画像の特定色を一括置換。
  • WebP / AVIF Converter
    画像形式の軽量変換。
  • FileType Sniffer
    ファイルの中身を解析して種別判定。

解析・変換ツール

データ処理・変換に役立つ技術系ツール群です。

  • JSON → Mermaid 変換
    JSON構造を図式化。
  • UnitMaster（単位変換ツール）
    数百種類以上の単位に対応し、工学・物理・生活用途まで高速変換。
  • Kanji Modernizer
    旧字体→新字体へ変換。
  • Old Kanji Reference
    旧字体辞書。
  • Redirect Unwrapper
    短縮URLの最終遷移先を表示。

生活・実用ツール

日常生活で役立つ軽量の実用ツールです。

  • JP Postal Lite
    郵便番号から住所検索。
  • 解約どこナビ
    サブスク公式解約リンク集。

その他ユーティリティ

ブラウザで動作する軽量ユーティリティツール。

  • WeatherDiff
    Open-Meteo・MET Norway 等を比較し予報のズレを可視化。地点入力・現在地対応。
  • WiFi Meter
    周囲Wi-Fiの混雑度を可視化。
  • Cover Letter Lite
    英文カバーレター生成。
  • Cosmetic Ingredient Checker Lite
    化粧品成分を簡易チェック。

運営方針

多くのツールは「ブラウザ内で完結する」ことを重視しており、 入力したデータはサーバーに送信せずローカルで処理されます。

また、NicheWorks ではすべてのツールにおいて 安全性・軽量性・ブラウザ完結型の動作 を重視しています。 個人情報の入力を必要とせず、ユーザーの手元だけで処理が完結するため、 セキュリティ面でも安心して利用できます。

今後も、小さな課題を解決するための実用的なツールを 静かに継続して追加していく方針です。

応援（寄付）

NicheWorks は個人が作っている小さなツール集です。 気に入ったツールがあれば、コーヒー１杯ぶんだけでも支援していただけるととても助かります。
