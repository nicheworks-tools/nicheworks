NicheWorks

「静かに役立つ」小さなWebツールの詰め合わせ

言語: 日本語 / English

このサイトについて

NicheWorks（ニッチワークス）は、
目立たないけれど確かに必要なものを、
小さく、静かに、かたちにしていくウェブプロジェクト群です。
派手さよりも実用性を、話題性よりも継続性を重視しています。

ツール一覧

  • ManualFinder（製品マニュアルリンク集）
    家電・カメラ・スマホなどの「公式マニュアル／取扱説明書ページ」だけを集めたリンク集。
  • TrashNavi（ごみ分別リンク集）
    自治体ごとの公式ごみ分別ページ・カレンダーへのリンク集。
  • 地味ログ整形屋（LogFormatter）
    Nginx / HTTP アクセスログをブラウザだけで整形・検索するツール。ファイルアップロード不要（ローカル処理）。
  • Rename Wizard（ファイル名整形ツール）
    ファイル名の一括整形用ツール。プレビューしてからコピーできるので安全にリネームルールを検討できます。
  • Contract Cleaner（契約書クリーナー）
    契約書テキストから、危険になりそうなキーワードを注視したい表現をハイライト表示する下読みチェック用ツール。
  • EXIF Cleaner Mini
    画像ファイルから EXIF 情報を削除する軽量ツール。ブラウザ内だけで処理し、画像はサーバーに送信されません。
  • Image Compression Inspector（画像圧縮率チェッカー）
    PNG / JPEG / WebP などの画像をアップロードして、ざっくりした圧縮率・品質と「どの形式が向いていそうか」を確認できます。
  • LineBreak Doctor（SNS改行修復ツール）
    X / Instagram / LINE などで崩れがちな改行を、投稿前に整えておくためのテキスト整形ツール。
  • Redirect Unwrapper（リダイレクト解除ツール）
    短縮URLや HTML メタリダイレクトを含むリンクを貼り付けて、最終的にどのURLへ飛ぶのかを確認するための補助ツール。
  • URL Title Collector（URLタイトル収集ツール）
    複数のURLを貼り付けると、各ページのタイトルを一括取得してメモや表にしやすくするツール。
  • 解約どこナビ（サービス解約リンク集）
    各種サブスク・サービスの「公式解約ページ」へのリンク集。別リポジトリで運用中。

運営方針

多くのツールは「ブラウザ内で完結する」ことを重視しており、 入力したデータはサーバーに送信せず、ローカルで処理されます。 一部ツールではアクセス解析と最低限の広告を表示する場合があります。
