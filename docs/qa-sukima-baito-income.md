# すきまバイト収入管理ツール 軽量QAチェックリスト

公開前の最低限の動線確認用チェックリストです（各項目 1 ケース以上）。

## 入力・編集・削除
- [ ] 手入力で追加 → 集計（年間/今月/月別）が更新される
- [ ] 既存データを編集 → 集計が更新される
- [ ] 既存データを削除 → 集計が更新される

## CSV
- [ ] CSV export → import で同等のデータが復元できる
- [ ] import 時に重複データが弾かれる（既存データとの一致）

## OCR
- [ ] OCR（一覧/詳細）→ プレビュー → 追加 → 集計が更新される
- [ ] 同一画像を2回OCRしても重複が弾かれる

## エラー表示
- [ ] エラー文が致命的に分かりにくくない（例: 選択なし/重複/不正入力の区別）
